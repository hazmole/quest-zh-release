遊俠閉起眼，把手放在樹幹上，學習關於森林的秘密。他們停下腳步，與當地的一隻松鼠聊天新朋友。他們訴說神秘，贏得陌生人的青睞。

如果想要扮演一個在文明邊疆活躍的熟練獵人和求生者，那麼遊俠會是個很棒的選擇。

{{< tab "故事與歌謠" >}}
<h3 style="color: var(--role-color-ranger);">故事與歌謠</h3>
{{< block >}}
〈鄉野樂談〉 — 〈民謠〉 — 〈講述傳說〉
{{< /block >}}
{{< ability-block title="鄉野樂談" type="ranger" >}}
{{< cost-tag number=1 >}}你使用化外之民的語言來博取鄰近距離內一個平民的信任。你必須創造出一個當地的俗語來與該 NPC 交流，像是「天空下起貓貓狗狗」或「不要用封面來評價一本書」。在遊戲桌上說明這句俗語的意思。

接著，你可以問以下任何問題，該 NPC 將會盡其所能地回答。
- 有任何人在鬧事嗎？
- 我可以去哪裡找到領導者？
- 大家最近都在談論些什麼？
{{< /ability-block >}}

{{< ability-block title="民謠" type="ranger" >}}
{{< cost-tag number=2 >}}你吟唱出能激起鄰近範圍內 NPC 們強烈感情的歌謠。（這首歌不會對敵對生物產生作用。）

在遊戲桌上朗讀或唱出這首歌的副歌段落，並描述你的表演。你可以撰寫你自己的歌謠、或是使用其他作曲者的作品。接著，選擇你在聽眾心中激發出了哪一種情緒：

**歡快**. 善意的熱情充滿心中，愉快歡樂的交談此起彼落。

**憂鬱**. 聽眾陷入沉默，開始對他們承受的痛苦進行哀傷的反思。

**驕傲**. 狂熱的情緒被點燃，使群眾們興奮並喧鬧了起來。
{{< /ability-block >}}

{{< ability-block title="講述傳說" type="ranger" >}}
{{< cost-tag number=3 >}}你透過講述一個當地的傳說故事來贏取一位平民的喜愛。你使用下列要素來編造一個傳說故事：

**義務**. 決定這個傳說故事會如何影響你的目標。選擇一項：
- 他會為你和你的同伴提供食物、住所、和他們家中的秘密。
- 他會為以最合理的價格來販售他所販售的一項物品。
- 他會去執行一個小任務，像是拿東西、或傳達一則消息。
- 他會承認一件最近的罪行。
- 他會原諒你的一個過錯。
- 他會散佈一個你編造的謠言。
- 一個你自訂的義務，只要它不會嚴重傷害到他就行。你必須擲骰來決定這會不會成功。

**教訓**. 根據你選擇的義務，說明這個傳說故事帶來了什麼教訓，像是「要為陌生人提供庇護」或「永遠做個公平的商人」。

**對象**. 指名一個中心人物或事件。

**故事**. 描述一個關於該人物或事件最有名的戲劇性情境。比如說一個為飢餓的人提供食物的傳教士，或者一場吞噬了眾多生命的大洪水。

在遊戲桌上將這個傳說故事講述給你的朋友們聽。你可以重複利用你已經創造出的傳說故事，不用每次都完整描述其內容細節。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "求生主義" >}}
<h3 style="color: var(--role-color-ranger);">求生主義</h3>
{{< block >}}
〈急救藥物〉 — 〈荊叢遮罩〉 — 〈救援信號〉— 〈神遊儀式〉
{{< /block >}}

{{< ability-block title="急救藥物" type="ranger" >}}
{{< cost-tag number=1 >}}你在短時間內搜尋附近區域來找尋用於治療一種疾病的草藥。你能找到剛好足夠製作一劑藥物的份量。這劑藥物能夠治療暫時性的疾病並排除毒素。
{{< /ability-block >}}

{{< ability-block title="荊叢遮罩" type="ranger" >}}
{{< cost-tag number=3 >}}你在野外露宿時請求周遭的植物群為整個隊伍提供保護。（鄰近範圍內必須要有植物生長。）灌木和荊棘藤蔓將會出現在營地周遭，將其隱藏起來。這個植物遮罩會持續直到你離開營地。

這個植物遮罩能把小型篝火的光源隱藏起來。如果有敵對生物靠近隊伍的營地，周遭的樹木將會沙沙作響、隨風呼嚎，帶來提前五分鐘的預警。其他生物可以強硬穿越這個植物遮罩，但這麼做將會使他受到１點傷害。當你在這個植物遮罩內「重振旗鼓」時，你的隊伍能改為獲得「休整」的好處。
{{< /ability-block >}}

{{< ability-block title="救援信號" type="ranger" >}}
{{< cost-tag number=4 >}}當處於野外時，你發出一個可以被其他遊俠看見的求救信號。描述你是如何發送你的信號：比如說升起狼煙、在樹上留下訊息、或其他類似的行動。

在第二天，你將會遇到一個前來協助你的 NPC 遊俠。出於對同事情誼的尊重，他會陪伴在你身邊，直到他完成你提出的一個請求。這個請求的內容不能毫無限制，且他不會永遠跟著你。

這位 NPC 遊俠擁有１０點生命值，攜帶著導引者所決定的裝備。你可以要求他做事，但他是一名獨立的角色，且最終會依據自己的興趣和信念行事。
{{< /ability-block >}}

{{< ability-block title="神遊儀式" type="ranger" >}}
{{< cost-tag number=4 >}}你在短時間從鄰近區域內找尋一單位的魔法食物。（該區域中必須要有植物生長。）

當你將其服用後，你在接下來一小時內將開始一趟心靈之旅，得到超自然的洞見能力。你可以得知關於下列其中一件事的真相：

- 前往某個地方最安全或最快速的途徑。
- 你的「宿敵」在過去一天內造訪過的地點。
- 你是否處於現實世界，或你認知的現實是否只是某種假象。
- 你的同伴是否有秘密瞞著你。（你會知道誰瞞著你，但不會知道秘密是什麼。）

如果祈喚衛士在使用他的〈通神祈喚〉能力時邀請了你，則你可以使用〈神遊儀式〉能力來加入他，一起參赴他前往星界位面的旅行。當你加入祈喚衛士的旅程時，〈通神祈喚〉能力的規則同樣會適用於你。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "尋路者" >}}
<h3 style="color: var(--role-color-ranger);">尋路者</h3>
{{< block >}}
〈解讀風向〉 — 〈導航〉 — 〈回聲探索〉— 〈與樹交談〉
{{< /block >}}

{{< ability-block title="解讀風向" type="ranger" >}}
{{< cost-tag number=0 >}}你專注在風的語言，感知你所在地區的天氣狀況。你能夠預測你所在地區在未來幾天的天氣狀況。<p>
{{< cost-tag number=2 >}}如果你花費２點 AP 在這個能力上，你可以決定你所在地區在未來幾天的天氣狀況，只要它符合時節即可。你可以選擇毛毛雨、雷雨、起霧、熱浪、或任何其他的天氣狀況，但不能選擇像是龍捲風或颱風之類的自然災害。
{{< /ability-block >}}

{{< ability-block title="導航" type="ranger" >}}
{{< cost-tag number=0 >}}你能在靈魂深處感受到大地的脈絡。除非受到魔法的妨礙，否則你在野外將永遠不會迷路。<p>
{{< cost-tag number=1 >}}當在野外旅行時，你帶領你的隊伍來到下列其中一個地區：

- **綠洲**. 你找到了一處可以喘息的地方，含有少量的食物和補給。
- **庇護所**. 你找到了一個天然洞窟，能夠你們躲避惡劣天氣的侵襲。
- **遺跡**. 你找到了不重要的廢墟遺跡，像是被遺棄的要塞或風化的雕像。
- **巢穴**. 你找到了一隻或一群動物的家園，像是水獺的巢穴。
{{< /ability-block >}}

{{< ability-block title="回聲探索" type="ranger" >}}
{{< cost-tag number=2 >}}你製造出聲響，讓其沿著天然洞窟系統或地下城之類的地下結構體傳播出去。（你可以拍掌、發出動物的聲音、或其他會引發回音的聲響。）你感知到與你所在區域相連的接下來三個區域的大致結構布局，以及任何連結著這些區域的通道的分布。導引者會為你畫一張簡易的地圖，標註任何重要的特徵，像是柱子、橋梁、或是裂隙。
{{< /ability-block >}}

{{< ability-block title="與樹交談" type="ranger" >}}
你觸碰一棵樹的樹幹，將你自己與相連森林的樹木群連結起來。

選擇一項：

{{< cost-tag number=2 >}} <b>搜索</b>. 你請求樹木為你搜索整座森林。舉例來說，你可以請它們為你定位一個特定生物、一個放置於某處的物體、或找尋一個地點，像是洞穴或建築。樹木會與它們的朋友交談，並接著告訴你目標的所在位置、怎麼過去、以及過去需要花多少時間。<p>

{{< cost-tag number=2 >}} <b>示警</b>. 你請求樹木守望你的隊伍。直到你離開這座森林之前，這些樹木將會在你們接近任何陷阱或危險生物時搖晃並沙沙作響。你們離危險越近，這個沙沙聲就會越大聲。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "獵人" >}}
<h3 style="color: var(--role-color-ranger);">獵人</h3>
{{< block >}}
〈追跡〉 — 〈狙擊〉 — 〈神射銳眼〉 — 〈無聲跟蹤〉 — 〈宿敵〉
{{< /block >}}

{{< ability-block title="追跡" type="ranger" >}}
{{< cost-tag number=0 >}}當你找到足跡時，你可以立即辨識出留下這些腳印的是什麼類型的生物。如果牠是動物，則導引者會告訴你任何你所知關於牠的情報，像是牠的習慣偏好或性情特徵。<p>
{{< cost-tag number=1 >}}即使附近沒有任何足跡，你也可以捕捉到一種動物所留下的蹤跡，只要牠是你當前所在環境的原生動物。你必須選擇一種動物種類，像是野豬或松鼠。（你不能使用這個能力來找尋特定生物。）你將在經過短暫的搜尋後找到該生物。
{{< /ability-block >}}

{{< ability-block title="狙擊" type="ranger" >}}
{{< cost-tag number=1 >}}你短暫的使你的感官變的更銳利，超越極限。你可以對一個位於遙遠距離，且你能看見的目標進行一次遠程攻擊，即使它超出射程之外也一樣。
{{< /ability-block >}}

{{< ability-block title="神射銳眼" type="ranger" >}}
{{< cost-tag number=0 >}}每當你在使用遠程武器攻擊時骰出了 20，你可以指定目標生物的一個特定肢體部位，並讓你的投射物命中該處。該肢體將會變得殘廢無法使用，直到有治療師將其治好。<p>
{{< cost-tag number=3 >}}你瞄準並專注在你的感官上，使用遠程武器攻擊鄰近距離內你所瞄準的一個生物或目標，並自動得到大成功的結果。這一擊將會使你瞄準的肢體殘廢。
{{< /ability-block >}}

{{< ability-block title="無聲跟蹤" type="ranger" >}}
{{< cost-tag number=0 >}}你和你的隊伍可以小心地跟蹤獵物而不會被發現，只要你們盡可能保持安靜。描述你和／或你們隊伍在狩獵跟蹤時的隊形。<p>
{{< cost-tag number=2 >}}當你找到你在追跡的生物或生物群時，你可以反過來埋伏牠們，即使牠們原本試圖埋伏你也一樣。你的隊伍進行第一輪的行動，且你獲得一個額外的回合。
{{< /ability-block >}}

{{< ability-block title="宿敵" type="ranger" >}}
{{< cost-tag number=4 >}}選擇一個你在過去曾遭遇過的特定生物。你把他視作你的宿敵。你同時只能有一個宿敵，且你可以在任何時候解除他宿敵的身分。

每當你的宿敵出現在場景中，你掠食者般的感官都能讓你立即察覺到他的存在，即使他隱藏起來了也一樣。當你的宿敵在鄰近距離內時，你的太陽穴會感到一陣刺痛。

當你和你的宿敵戰鬥時，你獲得下列額外的好處：
- 他無法埋伏並伏擊你。
- 你可以對他隨意使用你的〈狙擊〉能力而無須花費 AP。
- 當他在鄰近距離內時，即使他隱藏起來或處於隱形狀態，你也能掌握他的位置並使用物理攻擊攻擊他。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "自然之友" >}}
<h3 style="color: var(--role-color-ranger);">自然之友</h3>
{{< block >}}
〈與獸交談〉 — 〈動物夥伴〉 — 〈信使〉 — 〈搭檔羈絆〉 — 〈風中細語〉
{{< /block >}}

{{< ability-block title="與獸交談" type="ranger" >}}
{{< cost-tag number=1 >}}你觸碰一隻當前對你沒有敵意的動物，在接下來１分鐘內與牠建立一道心靈聯結。你能夠理解該動物的想法和感受，並可以和牠交流溝通。牠能夠提供關於各種事物的基本情報，像是牠感受到的、看到的、以及想要的，但牠無法理解複雜的概念和想法。
{{< /ability-block >}}

{{< ability-block title="動物夥伴" type="ranger" >}}
{{< cost-tag number=4 >}}你從鄰近地區馴服一隻願意盡其所能遵循你命令的動物夥伴。牠的體型不能比老鼠更小，也不能比馬還大。你同時只能有一個動物夥伴。

這隻動物是一個由導引者扮演的 NPC，並會在你的回合期間採取牠的行動。牠不會自願離開你所在的區域，且如果你們被拆散，牠會試圖去找你。牠不會說話，但牠能模糊地感覺到你的情緒和意圖。你的動物夥伴具有６點生命值，且可以進行會造成２點傷害的基本攻擊。
{{< /ability-block >}}

{{< ability-block title="信使" type="ranger" >}}
{{< cost-tag number=1 >}}你從鄰近地區找到一隻友善的動物並對牠低語，命令牠去遞送一個物品。這個物品的大小和重量必須要在該動物能合理攜帶的範圍內。你可以讓該動物把這個物品送到一個地點或一個特定人物手中。這個目的地必須要在牠一天可以旅行的距離內。
{{< /ability-block >}}

{{< ability-block title="搭檔羈絆" type="ranger" >}}
{{< cost-tag number=2 >}}你與你的動物夥伴之間發展出了獨特的心靈羈絆，這道心靈羈絆會持續到你招收了新的動物夥伴為止。你現在能被動地體驗到你動物夥伴的感受；你會瞥見牠所看到、聽到、和感覺到的事物。同時會在牠受到傷害時感受到一絲痛苦。

你可以在任何時候進入出神狀態，離開你原本的身體並直接控制你的動物夥伴。你仍然能模糊地意識到你原本身體周遭發生的事情。當處於出神狀態時，你可以如同牠就是你一樣控制你的動物夥伴，但你仍然無法說話。你可以在任何時候解除這個出神狀態。
{{< /ability-block >}}

{{< ability-block title="風中細語" type="ranger" >}}
{{< cost-tag number=4 >}}你低聲說出一個隨風飄往四面八方的口信。你必須要能清楚看見天空才能讓這個法術生效。這個口信會將一隻具有飛行能力的動物召喚到你的所在位置。你可以向任何類型的飛行動物發出請求，像是烏鴉或獅鷲。

這隻動物會在一小時之後抵達。只要你對牠表現友善，這隻動物就會在接下來一天內服從你的命令。舉例來說，你可以讓牠為你傳遞訊息、讓你騎乘、或干擾敵人。這隻動物不會主動攻擊敵人，除非敵人先攻擊牠。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "傳奇能力"  >}}
<h3 style="color: var(--role-color-ranger);">傳奇能力</h3>

{{< ability-block title="荒野名流" type="ranger" >}}
你的存在現在受到了世界各地荒野生物們的敬重。除了頭目以外的荒野動物從此不會再主動攻擊你，除非你先傷害牠們。（受過訓練的動物或寵物仍然能被命令去傷害你。）如果你已經學會〈與獸交談〉和〈與樹交談〉能力，則你現在可以直接使用它們而無須花費 AP。

你的〈動物夥伴〉能力現在能夠招收到更強大的動物。你的夥伴具有１０點生命值，攻擊能造成２點傷害，且每回合能夠攻擊二次。如果你已經擁有一個動物夥伴，則牠的數值將會一併被升級。

{{< cost-tag number=6 >}}當你在野外時，你發出呼喚以招來一群小型的動物軍團。牠們會在接下來這一天內跟隨你。你可以選擇任意的野生動物組合來集結成這個軍團，只要牠們能夠合理地來到這個區域。你可以命令牠們，且牠們會在你的回合期間共同行動。這個軍團被視作一個具有２０點生命值的單一生物，且牠們的攻擊能夠造成總共６點傷害，並被分散到最多６個目標上。
