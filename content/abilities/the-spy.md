間諜微笑，做出不可思議的招呼，滿懷自信地走進前門。他們在陰影中穿梭，在敵人疏於防備時發起突襲。他們製造出色的特製工具，讓他們在追求目標時獲得更多優勢。

如果你想扮演技藝純熟，依靠實際手段達成目標的角色，那你該選擇間諜。這同時也是扮演秘密特務或狡詐刺客的絕佳選擇。

<h3 style="color: var(--role-color-spy);">魅力</h3>
{{< block >}}
〈遊歷四海〉 — 〈舌燦蓮花〉 — 〈你不知道我是誰嗎？〉
{{< /block >}}

{{< ability-block title="遊歷四海" type="spy" >}}
你是精通世界各地文化研究的專家，尤其熟悉各個城市和國家的文化。當你謊稱自己是某個城市或國家的居民或官員，並因此受到質疑時，你可以表現出一個當地的特色習慣來欺騙普通人。

你必須想出一個特色習慣並在遊戲桌上描述其細節。這個特色習慣可以是手勢動作，像是特殊的握手方式、俗語、或是某種其他的風格特質。
{{< /ability-block >}}

{{< ability-block title="舌燦蓮花" type="spy" >}}
{{< cost-tag number=1 >}}如果你在違背禁令或做出不正當行為時被逮到，則你可以發揮機智來逃避懷疑。

你必須訴諸一個能讓對方認可的理由。導引者會從下列選擇一項作為正確答案，並把它秘密地寫下來：

- **命令**： 是某個權威人士要求你這麼做的。
- **無知**： 你什麼都不知道，這對你來說完全是一團亂！
- **賄賂**： 你給了某個有價值的東西來讓你矇混過關。

你必須猜測正確的方式是哪一個，並描述你是如何訴諸這個理由。

{{< cost-tag number=1 >}}每花費１點 AP，導引者就會為你排除一個不正確的選項。<p>
在你給出你的行動和理由後，導引者會揭露他先前寫下的選項。如果你猜對了，那麼對方就會放你一馬。
{{< /ability-block >}}

{{< ability-block title="你不知道我是誰嗎？" type="spy" >}}
{{< cost-tag number=1 >}}透過假裝成某個有名人，你讓平民和爪牙們留下深刻的印象。

要開始這個計畫，你首先要對一位隊伍成員使一個眼色。接著，每個你附近的隊伍成員都必須宣告一個能創造你身分的超猛事實。舉例來說，其中一個隊伍成員可能會先稱呼你是一位傳奇的作家、赫赫有名的戰士、或者大英雄，吸引大家的注意。隊伍的其他成員將會基於這個假身分繼續胡謅，直到所有人都貢獻了一份說法。每一次宣告都應該要比前一個還要更離奇。

當每個人都宣告完畢後，你將會吸引一大群鄰近範圍內生物聚集在你周遭，對你的事蹟深深著迷。

{{< tab "終結" >}}
<h3 style="color: var(--role-color-spy);">終結</h3>
{{< block >}}
〈偷襲〉 — 〈毒藥〉 — 〈死亡之手〉 — 〈懸賞〉
{{< /block >}}

{{< ability-block title="偷襲" type="spy" >}}
每輪一次，當鄰近距離內的某個敵人攻擊了一個除了你以外的生物，你可以趁機抓住他的破綻：<br/>
你立即移動到他身後，並進行下列其中一項行動：

{{< cost-tag number=1 >}}你對他進行一次基本攻擊。<p>
{{< cost-tag number=1 >}}你透過打擊這位平民或爪牙的穴道來使他失去行動能力。他將倒在地上並失去意識長達１０分鐘，或直到他被傷害。<p>
{{< cost-tag number=3 >}}你立即殺死這位平民或爪牙。描述你是如何解決他的。
{{< /ability-block >}}

{{< ability-block title="毒藥" type="spy" >}}
你運用從任何一家存貨正常的商店中買來的尋常物資，合成製造出一種毒藥。你製造出來的毒藥劑量只夠你使用一次。你可以將其塗抹在武器上、混入飲料或食物中、或直接讓一個生物服下。無論使用什麼方法，每一劑毒藥都只能影響一個生物。從以下任意選擇並組合出毒藥的效果：
{{< cost-tag number=1 >}}藍色. 受害者陷入沉睡，持續１小時。<p>
{{< cost-tag number=2 >}}白色. 受害者在極度痛苦中扭動，無法言語，持續１小時。<p>
{{< cost-tag number=3 >}}紫色. 受害者倒下，並在接下來的一天內癱瘓麻痺不能動彈。<p>
{{< cost-tag number=4 >}}黑色. 平民和爪牙會被立即殺死。頭目則會受到１０點傷害。<p>
{{< cost-tag number=6 >}}無色. 受害者不會產生任何症狀，但會在１周後死亡。（這個效果對頭目沒有效果。）
{{< /ability-block >}}

{{< ability-block title="死亡之手" type="spy" >}}
*魔法物品*. 一把單發式的手銃，使用無聲的魔法能量來推進彈頭。死亡之手能夠感測到你的目標，並引導你的手對場景內任何一個你能看見的目標進行一次完美射擊，包括位於遙遠距離的目標。這發射擊會造成６點傷害，並立即殺死平民和爪牙。
{{< cost-tag number=3 >}}你每次重新裝填這把武器都需要花費３點 AP 和一回合的時間。
{{< /ability-block >}}

{{< ability-block title="懸賞" type="spy" >}}
{{< cost-tag number=5 >}}你懸賞一個平民或爪牙的腦袋，要求將他活捉或殺死。除了這個能力的 AP 花費之外，你還需要額外提供一個有價值的物品作為獎賞，這個物品的價值至少要是珍稀等級。賞金獵人們將為了這個獎賞而參與競爭。

如果你要求活捉懸賞目標，你可以指定要把他抓到什麼地方。如果你要求殺死他，則你可以另外要求提供任何形式的證據，證明目標確實已經死亡。

{{< tab "隱蔽" >}}
<h3 style="color: var(--role-color-spy);">隱蔽</h3>
{{< block >}}
〈隱刃綁帶〉 — 〈偽裝〉 — 〈瞬現百寶袋〉 — 〈萬形劍柄〉 — 〈陰影斗篷〉
{{< /block >}}

{{< ability-block title="隱刃綁帶" type="spy" >}}
{{< cost-tag number=0 >}}除非你被另一個生物主動搜身，否則你可以在衣服裡藏起最多兩把武器而不會被注意到，只要你穿的衣服足夠把它們合理地隱藏起來就行。
{{< /ability-block >}}

{{< ability-block title="偽裝" type="spy" >}}
{{< cost-tag number=1 >}}任何時候當你在像是臥室、軍械庫、或辦公室之類的房間時，你可以找到各種能夠拼湊出一套可信偽裝的物品。根據你所在的地點，這套偽裝將會有至少一個能代表你假身分的標誌元素，像是制服、成員徽章、或是穿衣風格。這套偽裝能夠自動欺騙平民，但爪牙和頭目仍可能會質疑你。
{{< /ability-block >}}

{{< ability-block title="瞬現百寶袋" type="spy" >}}
*魔法物品*. 一個魔法郵差包，扣環上有一個隱藏的開關。當開關被打開，郵差包的開口就會變成一道通往陰影位面的傳送門，讓你能用來收納秘密物品。當開關被關閉，它的功能就跟普通的郵差包沒兩樣。收納在這個郵差包的間諜道具不會被算進你的道具欄上限中。（這讓你能夠攜帶超過１２個物品。）

這個郵差包與一個附有按鈕的魔法護符成對。

{{< cost-tag number=3 >}}你按下護符上的按鈕。無論瞬現百寶袋原本在哪裡，它都將傳送到你的手上。
{{< /ability-block >}}

{{< ability-block title="萬形劍柄" type="spy" >}}
*魔法物品*. 一把基礎武器（像劍或匕首）的握柄。
{{< cost-tag number=0 >}}如果你在持握萬形劍柄的時候想像著一把尋常武器，則這個劍柄會立刻變形成該武器。它在任何型態下都能在命中時造成２點傷害。當你不再需要這把武器時，它會變回原本劍柄的型態。
{{< /ability-block >}}

{{< ability-block title="陰影斗篷" type="spy" >}}
*魔法物品*. 一個能讓你除了影子之外的一切都隱形的魔法斗篷。
{{< cost-tag number=2 >}}當你啟動這個斗篷的能力時，它將短暫地閃爍發光，並讓你和所有你攜帶的東西消失不見。你在接下來１分鐘內變得隱形，但你在移動時的影子仍然會被映照出來。你的影子會在你保持不動時消失不見。

當處於隱形狀態時，除非你製造出很大的噪音，否則平民和爪牙甚至不會在你經過他們時察覺到你的存在。如果你在隱形狀態時發動攻擊，則這個斗篷的能力會失效，讓你重新變得可見。
{{< /ability-block >}}

{{< tab "監控" >}}
<h3 style="color: var(--role-color-spy);">監控</h3>
{{< block >}}
〈追蹤器〉 — 〈間諜眼鏡〉 — 〈訊息複製護符〉 — 〈探知寶珠〉
{{< /block >}}

{{< ability-block title="追蹤器" type="spy" >}}
*魔法物品*. 一組能讓你追蹤某個東西的裝置。

這套設備包括了一個魔法定位標籤，只有微型珠子的大小，且可以黏在大部分的物體表面上。追蹤器連接著一個魔法指南針，會顯示這個定位標籤所在的方向。

指南針的指針是黑色的。它會在魔法標籤與你出現在同一個場景時變成金色，並開始輕微的震動。
{{< /ability-block >}}

{{< ability-block title="間諜眼鏡" type="spy" >}}
*魔法物品*. 一副能讓你在自然黑暗環境中能如白晝視物的眼鏡。這副眼鏡可以被永久強化來獲得下列能力：

{{< cost-tag number=3 >}}增加能檢視鄰近距離內小型物體細節的放大功能、和能清楚看見遙遠距離事物的望遠功能。<p>
{{< cost-tag number=4 >}}增加紅外線視覺的功能，讓你能看見鄰近範圍內會產生熱量的事物的輪廓。這個能力可以穿透牆壁。<p>
{{< cost-tag number=6 >}}增加偵測某物是否為幻象的功能。被你偵測出的幻象會顯得有些扭曲且閃爍不定。

上述這些功能都可以在你戴上間諜眼鏡時，透過撥動裝置側面的旋鈕來各別開啟和關閉。
{{< /ability-block >}}

{{< ability-block title="訊息複製護符" type="spy" >}}
*魔法物品*. 一個讓你能竊聽通訊裝置的魔法護符。
{{< cost-tag number=0 >}}當你拿著這個護符待在某個能傳送訊息的法術或物體旁邊時，你可以把它傳送出去的訊息複製下來。你與訊息複製護符之間將建立一道心靈聯結，讓你自動在腦海中聽見被複製下來的訊息內容。你同一時間只能複製一個裝置的訊息；將訊息複製護符與另一個裝置配對將會中斷它與前一個裝置的聯結。
{{< /ability-block >}}

{{< ability-block title="探知寶珠" type="spy" >}}
*魔法物品*. 探知寶珠是一個跟你手掌差不多大的飄浮球體，並能夠透過心靈感應來回應你的命令。它的輪廓閃爍不清，幾乎看不見，但會在移動時發出低沉的嗡嗡聲。你可以命令探知寶珠移動到剛好離開你視線的位置或一個鄰近的房間中。它會告訴該空間中有多少生物存在，但不會告訴你這些生物是什麼種類。
{{< cost-tag number=1 >}}探知寶珠會給一張鄰近房間的簡陋心靈地圖。導引者會畫出該房間的簡易輪廓，強調任何重要的特徵（像是柱子或裂隙），並標記每個生物的確切位置。
{{< /ability-block >}}

{{< tab "速記" >}}
<h3 style="color: var(--role-color-spy);">速記</h3>
{{< block >}}
〈謄印魔書〉 — 〈仿錄妙筆〉 — 〈聽音石〉 — 〈密語之戒〉
{{< /block >}}

{{< ability-block title="謄印魔書" type="spy" >}}
*魔法物品*. 一本能複製文本內容的魔法書。

謄印魔書可以是任何尺寸──從口袋書到大型畫冊都可以。

你可以使用這本書來複製任何壓在它書頁上的文字或圖畫。舉例來說，如果你把一張手寫的信紙壓在謄印魔書的書頁上，書頁上就會出現一份清晰但不完美的複印副本。
{{< /ability-block >}}

{{< ability-block title="仿錄妙筆" type="spy" >}}
*魔法物品*. 一隻魔法筆，會引導你的手來複製任何作者的個人書寫風格。筆的另一端是一個魔法印章，能夠偽造任何正式印章。

只要把仿錄妙筆放到一份手寫文件的觸及距離內，它就會自動學會如何複製這份文件的手寫風格、簽名、和印章。

{{< cost-tag number=2 >}}你使用這隻筆讓它用過去曾學過的任何一種書寫風格偽造一份文件，包括官印或印章的複製。
{{< /ability-block >}}

{{< ability-block title="聽音石" type="spy" >}}
*魔法物品*. 一顆小型的魔法寶石，能夠被啟動來記錄鄰近範圍內的聲音，持續長達１個小時。如果你擁有〈訊息複製護符〉，則你可以將聽音石接到〈訊息複製護符〉上。當把它們相接時，你可以讓聽音石記錄〈訊息複製護符〉竊聽到的聲音。

這個裝置可以錄製長達１個小時的聲音。你可以輕輕擠捏它，使其播放它錄下來的聲音。使用這個裝置再次進行錄製，會讓它完全忘記過去記錄下來的內容。

{{< cost-tag number=1 >}}你啟動聽音石。它會錄製任何它聽到的聲音，並會特別留意於說話的聲音。
{{< /ability-block >}}

{{< ability-block title="密語之戒" type="spy" >}}
*魔法物品*. 一枚不透明的玻璃戒指，會在靠近字形符號時發光，例如書紙上的文字或石板雕文。
{{< cost-tag number=3 >}}你啟動密語之戒來翻譯或解讀密文。這枚戒指會閱讀任何你置於其觸及距離內的物體。解讀出來的翻譯文字會浮現在戒指周遭，並隨著你閱讀捲動。在它翻譯完畢後，這枚戒指就會自動停工。
{{< /ability-block >}}

{{< tab "滲透" >}}
<h3 style="color: var(--role-color-spy);">滲透</h3>
{{< block >}}
〈飄羽抓鉤〉 — 〈即興拼裝〉 — 〈萬用鑰匙〉 — 〈全視藍圖〉

{{< ability-block title="飄羽抓鉤" type="spy" >}}
*魔法物品*. 一個魔法抓鉤，會像氣球一樣慢慢地向上飄動，直到它飄到並抓住邊緣。你可以把任何長度的繩索綁在抓鉤上。
{{< /ability-block >}}

{{< ability-block title="即興拼裝" type="spy" >}}
{{< cost-tag number=1 >}}你的即興創作能力舉世無雙。<br/>
當你處在一個有各種日常用品能讓你使用的房間時，你可以在一分鐘內匆促做出以下其中一個物品：

- **絆繩警鈴**. 你製造出一個連接著聲音觸發裝置的絆繩，會在有人穿過它時發出響亮的噪音。
- **毒藥陷阱**. 如果你擁有毒藥，你可以製造出一個小巧的地板陷阱，讓第一個踩到它的生物中毒。
- **簡易武器**. 你製造出一把臨時武器，能夠在命中時造成２點傷害。
- **潛望鏡**. 你可以製造出一個小型潛望鏡，讓你能繞過有一點空隙的物體或門來視物。
{{< /ability-block >}}

{{< ability-block title="萬用鑰匙" type="spy" >}}
*魔法物品*. 一把能夠感應物理鎖的機械結構，並把自己完美變形成對應形狀的魔法鑰匙。它原本的型態是一把沒有齒形的簡易鑰匙。藉由轉動這把鑰匙的握柄，你可以把它變回原本的型態。
{{< cost-tag number=2 >}}這把鑰匙變形以完美對應一個新的鎖。

你每次讓鑰匙變形來對應一把新的鎖時都必須花費２點 AP。這把鑰匙會記住它過去曾解鎖過的鎖，且能夠不花費 AP 來變形成這些鎖的形狀。
{{< /ability-block >}}

{{< ability-block title="全視藍圖" type="spy" >}}
*魔法物品*. 一張魔法對開紙，能夠揭示出建築物的平面圖。

{{< cost-tag number=2 >}}你將全視藍圖壓在建築物的內牆上。魔法墨水會隨之浮現，揭示出這棟建築物中你當前所在樓層的整個格局。

導引者會為你畫一份這棟建築物的基本地圖，包含房間和走廊的形狀，以及門和窗戶的位置標示。全視藍圖只能作用在人造建築結構上。（它無法揭示出洞穴或其他天然環境的結構布局。）

如果你同時擁有〈探知寶珠〉，則全視藍圖可以即時顯示出它的所在位置，以及任何〈探知寶珠〉偵測到的鄰近範圍內生物的位置。

<h3 style="color: var(--role-color-spy);">扮演</h3>
{{< block >}}
〈偽造身分〉
{{< /block >}}

{{< ability-block title="偽造身分" type="spy" >}}
你謹慎地準備一個可信的替代身分。你的表面身分不能是一個實際存在的人物、也不能擁有特定的獨有職位，像是某個實際存在國家的掌權者。

每個表面身分都是由服裝、證件、和背景故事這三個要素構成。你可以創造並描述這三個要素。

當你使用你的替代身分時，你必須穿上該身分對應的服裝，把自己打扮成那個身分的樣子。

{{< cost-tag number=2 >}}你創造一個普羅大眾的表面身分，像是農夫、工匠、旅行商人、牧師、守衛、或教師。<br/>
當使用普通人的身分時，你可以毫不費力的融入人群之中，在公共場所往來而不被注意。
<p>
{{< cost-tag number=4 >}}你創造一個官員（擁有正式權威地位的人）的表面身分，像是政治家、法官、軍官、公會領袖、或船長。<br/>
當使用官員的身分時，你可以：

- 進入你的階級和隸屬關係允許你進入的限制場所。
- 給階級比你低的人下達普通命令，像是跑腿或放哨。
<p>
{{< cost-tag number=6 >}}你創造一個貴族的表面身分，像是著名的社會名流、富裕的企業家、使節、或地方行政長官。<br/>
當使用貴族的身分時，你可以：

- 進入富人或名流的專屬空間，像是私人俱樂部或舞會。
- 尋求當地統治者的接見。
- 花費１點 AP 來獲得能赦免你輕微罪刑的免罪證。

<h3 style="color: var(--role-color-spy);">傳奇能力</h3>

{{< ability-block title="暗影切割環" type="spy" >}}
*魔法物品*. 暗影切割環是一個可穿戴的手鐲，你能讓它伸展成一個大到足以讓你爬過去的環。
{{< cost-tag number=3 >}}當你把這個裝置壓到固體表面上時，你可以啟動它並創造出一個相同周長、且深度最多１公尺的洞。在使用過後，這個環會坍縮成原本的尺寸。

在被啟動時，它會發出一陣能被鄰近範圍內所有生物聽見的低沉的嗡嗡聲。這個裝置切下來的物質將會被瞬間傳送到某個隨機的陰影位面之中。
{{< /ability-block >}}

{{< ability-block title="奸巧假面" type="spy" >}}
*魔法物品*. 一個能投射出幻象，改變你外貌和聲音的小型面具。這個面具能夠紀錄所有鄰近範圍內你遇過的生物的外貌和聲音，並將其永遠記錄下來。
{{< cost-tag number=2 >}}你啟動這個面具，投影出一個逼真的幻象，讓你得到一個它記錄過的生物的外貌。它同時還會改變你的聲音，讓你聽起來就像是你冒充的生物一樣。它能讓你騙過除了冒充對象最親近的朋友和家人以外的所有人。
{{< /ability-block >}}

{{< ability-block title="夜行神探" type="spy" >}}
你是個擁有傳奇魅力的萬人迷，也是個聰穎過人的情報收集專家，這讓你能迅速收集到各種敏感的情報。
{{< cost-tag number=4 >}}當你滲透進某個人物的辦公室、營地、或家時，你能發掘出關於他的敏感情報。導引者從下面這張列表中選擇二項情報並揭示給你：

- 一個重要合夥人的身分。
- 一個它將會前往的地點，以及他前往的日期和時間。
- 一個一旦被揭露就會使他難堪的秘密。
- 他的財務狀況，包括他欠了誰最多錢、或是誰欠了他最多錢。
- 他對你或你隊伍的真實看法。
- 他最強烈的效忠對象，可能是對政府、行會、或其他組織。

導引者會決定你得到情報的具體細節。他也會決定是什麼文件揭露了這些情報，像是收據、日記、或信件。
