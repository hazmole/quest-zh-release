綠巫向花叢低語，鼓勵它們綻放芬芳。他化身為狼，大步在草原奔馳。他雙手高舉向天，喚來暴風雨，降下大自然的震怒。

若你想在這個世界探索與野生生物和原始元素力量的羈絆，你可以選擇扮演綠巫。

<h3 style="color: var(--role-color-naturalist);">變形者</h3>
{{< block >}}
〈動物形態〉 — 〈魚鰓術〉 — 〈轉鐵化芽〉 — 〈石化術〉 — 〈變形術〉
{{< /block >}}

{{< ability-block title="動物形態" type="naturalist" >}}
{{< cost-tag number=2 >}}你化身為野生動物的形態，得到牠的身姿、感官和行動方式。你可以選擇變身成任何體型介於老鼠和馬之間的生物。所有你攜帶的東西都會跟著變成你動物形態的一部分。

當處於動物形態時，你擁有６點生命值，並可以進行能造成２點傷害的基本攻擊。然而你不能在變形期間施放法術、使用道具、或說話。

你可以在任何時候變回你原本的形態。當你變回原形之後，你的生命值會變回你在變形之前的數值。如果你在變形期間讓生命值歸零，則你會立即變回你原本的形態。
{{< /ability-block >}}

{{< ability-block title="魚腮術" type="naturalist" >}}
{{< cost-tag number=1 >}}你對你持有容器內的少量清水施展魔法。當你將這些清水噴撒在一個生物身上，他會長出小巧但可見的魚鰓，讓他能在水下呼吸長達１小時。這個法術影響的清水量足以被使用６次。
{{< /ability-block >}}

{{< ability-block title="轉鐵化芽" type="naturalist" >}}
{{< cost-tag number=2 >}}你伸出手，閉上雙眼，將鄰近距離內一個金屬物體永久轉化成一叢繁盛的植物。（該物體不能具有魔法性質，且大小不能比一扇門還要大。）描述該物體變成了什麼樣的植物。
{{< /ability-block >}}

{{< ability-block title="石化術" type="naturalist" >}}
{{< cost-tag number=2 >}}你把鄰近距離內的一個平民或爪牙的體表變成石頭，持續１分鐘。目標在這期間將無法移動、視物、聆聽、或說話。該生物在被石化時擁有５０點生命。如果他的生命值在這個法術效果期間被歸零，則他會裂成碎塊並死亡。<p>
{{< cost-tag number=2 >}}你用硬殼包覆自己身上的盔甲或衣服。這層硬殼可以吸收最多１０點傷害。如果你被非魔法武器擊中，則該武器會裂成碎片。這層硬殼會在它受到傷害之後立刻粉碎消散。你可以在其他生物的回合中立即反應以施放這個法術來應對攻擊。
{{< /ability-block >}}

{{< ability-block title="變形術" type="naturalist" >}}
{{< cost-tag number=3 >}}這是你〈動物形態〉法術的進階版本。大部分規則都一樣，但有以下不同：

- 你可以改為選擇體型介於蒼蠅和大象之間的生物。
- 你在動物形態時具有１０點生命值。
- 你在動物形態時的攻擊改為造成３點傷害。
- 你可以用心靈感應的方式向隊伍成員傳遞訊息。
{{< /ability-block >}}

<h3 style="color: var(--role-color-naturalist);">召喚師</h3>
{{< block >}}
〈荊棘術〉 — 〈野餐寶盆〉 — 〈夜星之光〉 — 〈極光術〉 — 〈創世迴聲〉
{{< /block >}}

{{< ability-block title="荊棘術" type="naturalist" >}}
{{< cost-tag number=0 >}}你張開手掌，變出一個帶毒的荊棘刺針，直線射向鄰近距離內的一個目標生物或物體，造成２點傷害。
{{< rolling-tag >}}

||
| :-- |
| 20: 荊棘刺針的毒素被增強。如果你擊中一個有機生物，牠的皮膚會立刻腫脹起來，變得看不見觸及範圍外的事物，且直到他的下個回合結束之前都無法說話或施法。|
{{< /ability-block >}}

{{< ability-block title="野餐寶盆" type="naturalist" >}}
{{< cost-tag number=1 >}}你觸碰一個裝有食物、水、或油的容器。在接下來一分鐘內，這個容器會召喚出大量它原本盛裝的內容物，甚至讓多餘的物資豐盛地滿溢而出。舉例來說，如果你觸碰一個裝著水的陶壺，那麼水將滿溢出來並淹沒地板。或如果你觸碰一麻袋的穀物，那麼它將會撐開麻袋，堆積成小山。
{{< /ability-block >}}

{{< ability-block title="夜星之光" type="naturalist" >}}
{{< cost-tag number=2 >}}你在你頭頂正上方的高空召喚一縷星光。（你必須在室外才能施展這個法術。）在接下來一小時內，這縷星光會照亮大片區域，讓你可以看到很遠的地方。這縷星光會跟隨著你的移動，並總是坐落在你正上方的位置。這縷星光照亮的區域就彷彿是被日光照射一樣明亮。
{{< /ability-block >}}

{{< ability-block title="極光術" type="naturalist" >}}
{{< cost-tag number=5 >}}隨著你的雙眼如舞動星火般閃爍，你在正上方的天空中召喚出耀眼炫目的虹色極光。這道極光會持續１０分鐘，且最多一公里內的所有生物都會看見它。

**平民**：所有能看見極光的平民都會感到目眩神迷，停下手中的工作以盯著它看。

**爪牙**：爪牙同樣會被這個法術給弄到目眩神迷，並會停下他們正在做的事以盯著它看。擊中爪牙將會使他擺脫這個法術的影響。

**頭目**：導引者必須擲骰。若擲出 2-20，則頭目會因極光而目眩神迷，且必須在他的回合中從移動和行動中選擇。若擲出 1，則他會呆在原地，持續直到這個法術結束或他被擊中。
{{< /ability-block >}}

{{< ability-block title="創世迴聲" type="naturalist" >}}
{{< cost-tag number=6 >}}你哼唱出迴盪的旋律，召喚一群從創世之時就已經存在的魔法星火。

星火曾經是強大的光靈，在遙遠的過去中漫游穿梭在宇宙諸界之中。然而，如今這些縹緲的靈質殘留體就彷彿垂死的群星，燃燒著它們最後的能量。它們不再具有智能和意識，但似乎仍能感應到它們周遭的環境。

在接下來一分鐘內，這群星火會在這個區域中飄盪，恢復所有生物的生命值，並賦予植物新生。場景中的每個生物都會在星火在場時停止他們正在做的事情，靜止不動，懷著敬畏的心情，直到它們消散不見。

{{< tab "元素使" >}}
<h3 style="color: var(--role-color-naturalist);">元素使</h3>
{{< block >}}
〈冰凍〉 — 〈灼燒〉 — 〈雷電〉 — 〈火球術〉
{{< /block >}}

{{< ability-block title="冰凍" type="naturalist" >}}
{{< cost-tag number=0 >}}你吐出冷空氣，製造出冰冷刺骨的寒風席捲鄰近距離內的一個生物或物體。受到影響的生物將會感到一股深及骨髓的寒意，並被造成１點傷害。你可以使用這個法術來熄滅小型的火焰。<p>
{{< cost-tag number=2 >}}你凍結鄰近距離內的一個平民、爪牙或物體，將其裹在冰中，直到你從該場景中離開。如果你凍結的是生物，那麼他將無法移動或行動。當該生物受到傷害時，冰塊將會碎開並讓目標被釋放。
{{< /ability-block >}}

{{< ability-block title="灼燒" type="naturalist" >}}
{{< cost-tag number=0 >}}你吐出熱空氣，製造出焦灼燥熱的熱風席捲鄰近距離內的一個生物或物體。這股熱度足以讓該生物感到強烈不適，但並不會造成傷害。<p>
{{< cost-tag number=1 >}}你灌注了更多能量在這個法術上，並改為使目標點燃起火，對其造成２點傷害，使其熊熊燃燒起來。
{{< /ability-block >}}

{{< ability-block title="雷電" type="naturalist" >}}
{{< cost-tag number=4 >}}選擇一個你能看見的生物或物體作為目標。一道閃電從天而降、或從你的雙手迸射而出，對目標造成８點傷害，並製造出震耳欲聾的雷鳴巨響。如果該生物的身體與金屬物質相連，則金屬會因此融化，對目標造成額外的２點傷害。
{{< /ability-block >}}

{{< ability-block title="火球術" type="naturalist" >}}
{{< cost-tag number=5 >}}你在你面前形塑出一團滯留在半空中的火焰球體，並讓它直線衝向你能看見的一個目標。這團火球會在接觸時爆炸，並對任何它擊中的生物和物體造成６點傷害。你可以控制這個法術避開任何你不想傷害的盟友或生物。
{{< rolling-tag >}}

{{< tab "暴風御者" >}}
<h3 style="color: var(--role-color-naturalist);">暴風御者</h3>
{{< block >}}
〈喚霧術〉 — 〈漩渦術〉 — 〈狂風術〉 — 〈激流術〉 — 〈暴風呼喚〉
{{< /block >}}

{{< ability-block title="喚霧術" type="naturalist" >}}
{{< cost-tag number=1 >}}你的手在空中攪動，製造出一團以你為中心的濃霧。這團濃霧會迅速的擴散開來，順著轉角和下坡蔓延。僅僅數秒的時間，鄰近範圍內的一切都將籠罩在濃霧之中。處於濃霧中的生物將無法看見自己觸及範圍以外的事物，使他們無法進行遠程攻擊。這團濃霧會在你離開該區域、或者該區域被強風襲擊時消散。
{{< /ability-block >}}

{{< ability-block title="漩渦術" type="naturalist" >}}
{{< cost-tag number=2 >}}你在你能看見的水體中製造出一陣呼嘯疾轉的漩渦。這陣漩渦會持續存在直到你離開該區域。任何接觸到這陣漩渦的生物都會被猛烈的吸入、淹沒、然後被拋出，並受到２點傷害。
{{< /ability-block >}}

{{< ability-block title="狂風術" type="naturalist" >}}
{{< cost-tag number=3 >}}你閉上眼，攪動空氣，形塑你周遭的風。一股狂風成形，你可以在該場景中驅使它席捲任意大小和形狀的區域，只要你能夠看見被影響的區域就行。在影響範圍內，這股狂風會以猛烈的高速疾馳，揚起沙塵，並把小型的物體捲飛到空中。狂風中生物的視線受阻，只能看見鄰近範圍內事物。

當你開始施展這個法術時，這股狂風會將生物撞倒在地，把他們往後吹飛滾動，直到他們離開狂風的影響範圍。處於狂風影響範圍外的生物將無法進入其中，除非他的體型比影響範圍還大。
{{< /ability-block >}}

{{< ability-block title="激流術" type="naturalist" >}}
{{< cost-tag number=4 >}}你對著遠程距離內一條流動的河流低聲吟誦出魔法咒語。一股力量將匯聚在河流的上游。１５秒後，一陣怒濤激流將沿著河道奔湧到你所在的位置。所有被捲入激流中的生物都會被沖到下游，擱在遠超過你視線範圍外的遙遠河岸上。
{{< /ability-block >}}

{{< ability-block title="暴風呼喚" type="naturalist" >}}
{{< cost-tag number=2 >}}你在空中你能看見的某處製造出一大團漆黑帶電的烏雲。這陣暴風雨將會持續存在直到你從該場景離開。<p>

在這陣暴風雨存在的期間，你可以呼求它製造出以下這些效果：
{{< cost-tag number=1 >}}你製造出一陣傾盆大雨。<p>
{{< cost-tag number=1 >}}一聲震耳欲聾的雷鳴震撼著大地。所有在暴風雨下的生物直到該場景結束之前都聽不到任何他觸及距離外的聲音。<p>
{{< cost-tag number=3 >}}如果你已經學會〈雷電〉能力，則你可以只花費３點 AP 來使用它。<p>
{{< cost-tag number=x >}}一道連鎖閃電劈穿你能看見的 NPC 們，對他們各自造成４點傷害。每個目標都需要花費１點 AP。
{{< /ability-block >}}

{{< tab "精魂求喚者" >}}
<h3 style="color: var(--role-color-naturalist);">精魂求喚者</h3>
{{< block >}}
〈野性風貌〉 — 〈獵物警覺〉 — 〈自然之眼〉
{{< /block >}}

{{< ability-block title="野性風貌" type="naturalist" >}}
{{< cost-tag number=1 >}}鷹. 你在接下來一分鐘內獲得鷹眼般的視力。你能清楚看見你視線範圍內的任何東西，即使它處於遠處也一樣。你也可以察覺到一般人感覺不到的遠方傳來的動靜。<p>
{{< cost-tag number=2 >}}貓. 你發出輕柔的呼嚕聲，向鄰近距離內最多６個生物注入貓的精魂。在接下來一小時內，受影響的生物將獲得在完全黑暗中仍能看見周遭環境的能力，就彷彿他們身處在昏暗的房間一樣。這個效果無法看穿魔法黑暗。受影響的生物可能會偶爾感覺到一股衝動，停下他們正在做的事情並清潔自己。<p>
{{< cost-tag number=2 >}}狼. 你發出嚎叫，對鄰近距離內最多６個生物賦予迅捷之力。（你必須在遊戲桌上嚎叫。）在接下來一小時內，受到這個法術影響的生物在用步行移動時能跑得比其他任何生物都快。
{{< /ability-block >}}

{{< ability-block title="獵物警覺" type="naturalist" >}}
{{< cost-tag number=1 >}}你增強你的危機意識，持續一個小時。每當你即將要受到其他生物給直接威脅時，你的身體會感應到一股微妙的信號。（比如說你的寒毛豎立、或可能感到一股刺骨的不安。）包括那些你看不到的生物。

如果你透過這個方式偵測到生物，那麼你的隊伍就不會被伏擊。（如果你查覺到某個生物試圖攻擊你們，那麼你的隊伍將可以在戰鬥中採取第一輪行動。）
{{< /ability-block >}}

{{< ability-block title="自然之眼" type="naturalist" >}}
{{< cost-tag number=3 >}}你延展你的感官，持續一個小時。選擇二項：

- **靈光感應**. 你能感應到一陣微弱的輪廓環繞在任何正受到法術影響的事物身上。
- **紅外視覺**. 你能感應到鄰近範圍內，與環境的相對溫度較為明顯的事物，像是有著溫暖身體的生物。這個感官能夠穿透牆壁。
- **黑暗視覺**. 你能在黑暗中能看見鄰近範圍內的事物，就如同你處在昏暗照明的環境中。
- **鏡折視覺**. 你能看穿封閉空間內（像是建築物）轉角後方的景象。你必須處於該轉角位置的觸及距離內。<br/>（舉例來說，你可以看見走廊拐彎處後的事物。）
- **真實視覺**. 你能感應到某個事物是否為幻象。透過這個法術感應到的幻象將顯得閃爍搖曳。
{{< /ability-block >}}

{{< tab "生態學家" >}}
<h3 style="color: var(--role-color-naturalist);">生態學家</h3>
{{< block >}}
〈命令植物〉 — 〈石之記憶〉 — 〈季節轉移〉
{{< /block >}}

{{< ability-block title="命令植物" type="naturalist" >}}
{{< cost-tag number=0 >}}你對周圍的活體植物低語一段押韻的對句。你可以自己寫一段，或直接借用其他作者的。使用該株植物可能會喜歡的詩句，像是「聽我天籟、恭迎春來」或「花如解語迎人笑，草不知名隨意生」。

你簡短的詩句將能請求植物透過輕微的移動或生長來產生無害的效果。你想要操控的植株必須在遠程距離內。你可以讓樹木沙沙作響、讓葉片掉落、讓植物開花、讓藤蔓生長、或讓種子發芽。
{{< cost-tag number=1 >}}如果你能夠合理描述過程，你就能讓植物做出對鄰近距離內的一個目標造成２點傷害的效果。例如：你可以讓藤蔓長出棘刺並鞭打一個生物、或讓樹木搖晃來把橡實狠狠砸在目標的頭上。
{{< /ability-block >}}

{{< ability-block title="石之記憶" type="naturalist" >}}
{{< cost-tag number=3 >}}如果你遇到一塊歷經風霜的岩石，像是一座山、峽谷、或雕像，你可以觸碰它來引導其古老的智慧。你必須對這塊岩石講述關於你家族的故事——也許這會是一段家族團聚的記憶，比如說節慶時分。在遊戲桌上分享這個故事。

這塊岩石遺老會用它的智慧來回報你的故事：
- 你能感知身處的世界是否為真實且自然的，還是某種陰謀幻象。
- 你能感知地表下是否存在著任何失落的遺跡，像是隱藏的神殿或古老的墓穴。
- 你模糊地感知到世界上最為強大的失落神器，以及它可能所在的地區。

<h3 style="color: var(--role-color-naturalist);">傳奇能力</h3>

{{< ability-block title="自然之怒" type="naturalist" >}}
{{< cost-tag number=7 >}}你引導元素狂怒，創造出一陣以你為中心，半徑１公里的暴風雨。只要你選擇繼續引導，這個效果就會一直持續，但是你在專注於維持這個法術的期間，每回合都會受到１點 HP 的傷害。你可以決定這陣暴風雨的暴風眼的大小，裡面是安全的。你在每次施放這個法術時決定其災難的性質：

**閃電**. 閃電在這陣暴風雨中肆虐，使電弧在導電物體中流竄、並轟擊任何被捲入其中的 NPC。在你施放這個法術的期間，每個被捲入這陣暴風雨的 NPC 每回合都會受到２點傷害。

**暴雪**. 水氣在這陣暴風雨中凝結，形成一顆顆蘋果大小的冰雹。暴風雨區域中的溫度寒冷刺骨。在你施放這個法術的期間，每個被捲入暴風雨的 NPC 每回合都會受到１點傷害。在冰雪風暴的肆虐之下，生物將無法看見超過他們觸及距離以外的事物。

**颶風**. 挾帶著駭人威力的狂風在暴風雨中呼嘯狂掃。脆弱的建築物將被徹底摧毀，所有的窗戶都會被粉碎，大量的屋頂將會塌陷，且樹木可能會被連根拔起。大部分未被固定的物體都會被風暴捲起，並吹飛到很遠的地方。在你施放這個法術的期間，任何被捲入暴風雨中的 NPC 都會被吹得七葷八素，無法移動，且每回合受到４點傷害。
{{< /ability-block >}}

{{< ability-block title="荒野進化" type="naturalist" >}}
你的肉體本質與你所選擇的一種動物型態將永久地交織在一起。這種動物的體型必須介於老鼠和大象之間。這個型態只能選擇一次。

你可以在你的原始型態和你所選動物型態之間隨意變化。當你改變型態時，你仍會維持你當前的生命值。

你在變形期間將獲得該種動物的天生能力。舉例來說，如果你變成了一隻老鷹，那麼你就能飛行。你在動物型態時的基本攻擊能夠造成３點傷害。

你現在可以在處於動物型態時施放法術，並能夠用心靈感應和你的隊友溝通。

你仍然可以使用〈動物形態〉和〈變形術〉能力，且這些能力的規則仍然如常適用。
{{< /ability-block >}}

{{< ability-block title="塵歸塵" type="naturalist" >}}
{{< cost-tag number=3 >}}你讓一個人造物體回歸自然，瞬間使它化作塵土。

這個物體必須要小到足以裝進一個１０公尺立方的立方體空間內。這個變化是永久的，且無法被逆轉。
{{< /ability-block >}}

{{< ability-block title="新世界祈願" type="naturalist" >}}
你成為了新世界的種子。施放這個法術將會永遠耗盡你的生命；你再也無法被復活。

你對大自然的全部知識都將在這個法術中體現。當你施放這個法術時，你的身體會在光輝中消散，變成一道奔騰的能量波，徹底改變你所身處的星球。這道能量波首先會改變大氣層，如果它原本不適宜生命維持，則大氣層將變得能夠維持生命運作。

接著，一叢叢生機盎然的植物群將會從你原本的位置迸發，迅速生長並覆蓋整個星球的表面。這陣繁盛的植物生長會以你的位置為起點向外擴散，持續直到整個星球都被轉化。如果海洋和河流已經不復存在，則這個法術將會把它們重新創造出來。

你可以在施法時決定讓這個法術可以分解所有的人造建築，像是各種建築設施，讓大地回歸自然的懷抱。所有因此被摧毀的物體都會被溫柔優雅的方式崩解，讓原本在其中的生物，像是人類和動物，能毫髮無傷地落到地面。

這個變化是永久的，且無法被魔法反制手段給阻止、中斷、或逆轉。
