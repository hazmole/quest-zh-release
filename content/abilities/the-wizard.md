法師誦唸強大的咒語，釋放出力場衝擊波擊退面前的一切阻礙。他們能在緊要關頭憑空召喚出需要的東西；藉著舞動雙手，他們能夠開啟通往未知維度的裂口，探求超越時空的真理。

你可以使用法師這個靈活多變的角色來扮演各種類型的魔法使用者，從勤奮研究法術書的智者到追求絕對力量的狂妄巫師。

{{< tab "塑能" >}}
<h3 style="color: var(--role-color-wizard);">塑能</h3>
{{< block >}}
〈魔法轟擊〉 — 〈燃火術〉 — 〈雷鳴波〉 — 〈念動〉 — 〈最後曙光〉
{{< /block >}}

{{< ability-block title="魔法轟擊" type="wizard" >}}
你必須拿著一個帶有尖端的物體才能使用這個法術，像是法杖、魔杖、或寶劍。你使用它朝著你可以看見的一個目標直線射出一顆閃爍著的力場飛彈，造成２點傷害。（進行擲骰；這個行動視為一次基本攻擊。）
{{< cost-tag number=2 >}}你讓力場飛彈的能量超載，把它造成的傷害提高到４點，且無須擲骰就自動命中目標。
{{< /ability-block >}}

{{< ability-block title="燃火術" type="wizard" >}}
{{< cost-tag number=0 >}}你摩擦雙手，點燃鄰近距離內一個你能看見的可燃物體。這個法術可以點燃紡織品、木材、紙張、和其他可燃材料。

這個法術不能直接用在生物身上，但它可以被用來點燃他們穿著或攜帶的可燃物。點燃生物身上的東西會對他造成１點傷害。
{{< /ability-block >}}

{{< ability-block title="雷鳴波" type="wizard" >}}
{{< cost-tag number=2 >}}你一拍雙掌，朝著你面對的方向釋放出一陣雷鳴震波。這陣衝擊波會擊中鄰近距離內的最多３個生物，並造成２點傷害。如果被影響生物的體型與馬相近或更小，則他們會同時被擊倒在地。
{{< /ability-block >}}

{{< ability-block title="念動" type="wizard" >}}
你使用你的心靈力量來移動一個鄰近距離內的生物或物體。

{{< cost-tag number=1 >}}你可以移動一個大小不超過大象的物體或生物。如果你和緩地移動他，那麼你可以控制他最多１０秒的時間。或者，你可以猛力把他投擲到最遠５公尺的距離，並造成２點傷害。<p>
{{< cost-tag number=x >}}你可以花費額外的 AP 來增強這個能力的強度。你每花費１點 AP，就可以在投擲該生物時讓最大距離增加５公尺，並使他受到的傷害增加２點。（舉例來說，如果你花費５點 AP，你可以把目標丟到最多２５公尺的距離，並造成１０點傷害。）
{{< /ability-block >}}

{{< ability-block title="最後曙光" type="wizard" >}}
你必須拿著一個帶有尖端的物體才能使用這個法術，像是法杖、魔杖、或寶劍。

{{< cost-tag number=3 >}}你瞬間引導一道炫目的集束強光直射向一個你能看見的目標，造成６點傷害。這道光束會點燃任何它接觸到的可燃物、在鐵製品上熔出一個洞、並燒焦有機物質。<p>
{{< cost-tag number=5 >}}你讓這道光束的能量超載，它將立即殺死平民和爪牙，或對頭目造成１２點傷害。如果〈最後曙光〉殺死了目標，那麼它會被分解成塵土。
〈最後曙光〉創造出的光束會被鏡面給反射。
{{< /ability-block >}}

{{< tab "召咒" >}}
<h3 style="color: var(--role-color-wizard);">召咒</h3>
{{< block >}}
〈魔寵〉 — 〈天馬斗篷〉 — 〈力場牆〉 — 〈憑空召喚〉
{{< /block >}}

{{< ability-block title="魔寵" type="wizard" >}}
{{< cost-tag number=2 >}}你召喚出一隻體型小巧的靈體生物，像是鳥、蜥蜴、蝴蝶、或任何其他類似大小的生物。這個魔寵是一個你所在世界中的隨機靈體生物的化身。他將成為你的盟友，並會遵循你的指示。導引者會決定這隻生物的背景和人格特質，並負責扮演他。

如果魔寵被虐待，則他可以選擇離開你。除此之外的情況，這個魔寵將無法移動到你視線範圍之外的位置，且只能撿起或移動體積不超過錢包，且重量不超過１０磅的東西。他無法進行攻擊。

你的魔寵具有４點生命值，且免疫所有非魔法的傷害。如果他的生命值歸零，則他將消失不見，且你們之間的連繫也將被打破。你同時只能有一隻魔寵。
{{< /ability-block >}}

{{< ability-block title="天馬披風" type="wizard" >}}
{{< cost-tag number=3 >}}你賜予鄰近範圍內任意數量的生物一襲幻影般的羽毛披風，持續１個小時。受到影響的生物將會漂浮在離地１公尺的位置，並可以朝著任何他們想要的方向滑翔移動。如果他們離地的高度超過１公尺，則他們會緩緩飄落直到到達該高度。
{{< /ability-block >}}

{{< ability-block title="力場牆" type="wizard" >}}
{{< cost-tag number=3 >}}你創造出一道薄如紙的力場牆，能夠阻止任何非魔法的生物和物體穿越。你可以選擇要讓這道力場牆阻擋單向或雙向的通行。這道力場牆可以呈現任何你想要的形狀，但它的整體大小必須要能夠塞入一個１０公尺邊長的立方體內。這道力場牆擁有２０點生命值，並會在生命值歸零時消散。

你可以讓這道力場牆靜止不動、或利用心電感應移動它。（舉例來說，你可以在腳下創造出一個隱形的平台，在你行走時隨著你一起滑動。）
{{< /ability-block >}}

{{< ability-block title="憑空召喚" type="wizard" >}}
{{< cost-tag number=3 >}}你想像一個物體，並使它憑空出現在鄰近距離內的某處。你可以召喚出任何滿足下列條件的物體：

- 它的大小不能比大象更大。
- 它不能具有魔法效果。

你必須告訴導引者你想要召喚什麼物體。接著，導引者會想出一個關聯的物體，並秘密告訴你它的名字。（舉例來說，如果你想要召喚一個撬鎖工具，則導引者可能會要求你畫一扇門。）

設定一個計時器，你有１分鐘的時間畫出這個關聯物體。時間到之後，把你畫出的成品展示給你的隊伍看。你的朋友們可以討論他們看到的東西，但他們只有一次猜測的機會。不要給隊伍除此之外的線索。如果他們猜對，那麼你想像的物體就會憑空出現。

如果他們猜得足夠接近但並不準確，該物體仍會出現，但帶有缺陷。如果他們完全猜錯，那麼會改為出現一個隨機的畸形物品。
{{< /ability-block >}}

{{< tab "位面轉移" >}}
<h3 style="color: var(--role-color-wizard);">位面轉移</h3>
{{< block >}}
〈閃現術〉 — 〈歸還錨點〉 — 〈傳送門〉 — 〈黑暗門扉〉 — 〈瞬間傳送〉
{{< /block >}}

{{< ability-block title="閃現術" type="wizard" >}}
{{< cost-tag number=0 >}}你瞬間移動到鄰近距離內一個你所選擇的位置。你會原地消失，括起一陣輕柔的微風後，接著出現在該位置。
{{< /ability-block >}}

{{< ability-block title="歸還錨點" type="wizard" >}}
{{< cost-tag number=1 >}}<b>定錨</b>. 你把自己跟你當前所在的房間綁定。這個房間必須要屬於你或其中一個隊伍成員。這個房間現在變成了你的「歸還點」。你同時只能把自己綁定於一個歸還點。<p>
{{< cost-tag number=3 >}}<b>歸還</b>. 你打個響指，任何鄰近範圍內且自願的隊伍成員將立即被傳送到你設定的「歸還點」。
{{< /ability-block >}}

{{< ability-block title="傳送門" type="wizard" >}}
{{< cost-tag number=3 >}}你召喚出一個任意形狀和面向，且大小不能超過一扇門的傳送通道。它出現在鄰近距離內一個你所選擇的位置。這個傳送門看起來像是一個通往另一側的窗口，邊緣劈啪閃爍且時不時有能量迸發而出。當你再次施放這個法術時，新的傳送門將會自動連接到前一個傳送門。傳送門會維持開啟的狀態一直到你決定將其關閉。任何能夠塞進這個傳送門的事物都會抵達另一側。
{{< /ability-block >}}

{{< ability-block title="黑暗門扉" type="wizard" >}}
{{< cost-tag number=2 >}}你輕敲一扇門，使它成為通往某個特定陰影位面的臨時傳送門。你必須要知道該陰影位面的存在才能夠創造出通往該處的門扉。（舉例來說，如果你看見某個生物帶著「次元收納袋」，則你可以開啟一道通往該收納袋中陰影位面的傳送門。）

當你打開這扇門時，你可以看見存在於該位面中的事物，且你和你的盟友們可以進入其中。如果這扇門被關上，傳送門的效果也將隨之消失。（如果這扇門在你身後被關上，則你將被困在陰影位面中，且必須透過其他方式逃脫，像是〈歸還錨點〉。）
{{< /ability-block >}}

{{< ability-block title="瞬間傳送" type="wizard" >}}
{{< cost-tag number=5 >}}你打了個響指，你和鄰近範圍內任意數量的自願生物將瞬間消失，留下一小陣衝擊波後，傳送到任何你選擇的地方。

{{< tab "巫術" >}}
<h3 style="color: var(--role-color-wizard);">巫術</h3>
{{< block >}}
〈法術禁令〉 — 〈反射〉 — 〈抄寫卷軸〉 — 〈法術竊取〉
{{< /block >}}

{{< ability-block title="法術禁令" type="wizard" >}}
{{< cost-tag number=2 >}}透過說出「禁止」這個詞，你在鄰近距離內一個你能看見的 NPC 施法時嘗試無效他的法術效果。你必須在導引者描述該法術的結果之前宣言你要使用這個能力。

{{< rolling-tag >}}

||
| :-- |
| 20: 該法術被猛烈中斷，且其施法者被擊中而失去１點生命值。|
| 11-19: 該法術在施放前就被阻止。|
| 6-10: 該法術打失，轉而命中另一個鄰近距離內的生物或目標。|
| 2-5: 該法術如常發揮原本的效果。|
| 1: 你意外增幅了該法術的效果。如果該法術是傷害法術，則它的傷害加倍。|
{{< /ability-block >}}

{{< ability-block title="反射" type="wizard" >}}
{{< cost-tag number=3 >}}你編織出一個反制法術，把一個以你為目標的法術反彈。你必須在導引者說你成為了該法術的目標時才能宣言你要反射該法術。

{{< rolling-tag >}}

||
| :-- |
| 11-20: 施法者的法術擊中了他自己，而不是你。|
| 6-10: 法術被你反彈開來，但也沒有擊中其施法者。|
| 1-5: 法術被你反彈開來，改為擊中了一個鄰近距離內的隊伍成員。|
{{< /ability-block >}}

{{< ability-block title="抄寫卷軸" type="wizard" >}}
{{< cost-tag number=2 >}}你製造出一張「法術卷軸」，能夠被用來施放一個你知曉的法術。你使用〈抄寫卷軸〉時的 AP 花費必須額外加上你所抄寫法術的 AP 花費。舉例來說，如果你製造出一張能施放〈瞬間傳送〉法術的魔法卷軸，那麼你必須要花費總共７點 AP。

你必須在一個安全且安靜的場所花費一小段時間來製備卷軸。由你決定捲軸上寫了什麼。卷軸的使用者必須要大聲念出卷軸上的文字才能啟動其中儲存的法術。
{{< /ability-block >}}

{{< ability-block title="法術竊取" type="wizard" >}}
{{< cost-tag number=3 >}}在你發現鄰近距離內的一個生物正在施法時，你可以短暫潛入他的腦海中以瞥視其本質。你學會該法術，並可以施放它一次。

你竊取到的知識會給你最低限度的理解，讓你知道該如何重現該法術，但無法讓你精通它。導引者可以選擇不揭示某些法術的完整效果或潛在後果。

一旦你施放了偷來的法術、或使用〈抄寫卷軸〉把它抄寫下去，你就會遺忘該法術並因此無法再使用它。
{{< /ability-block >}}

{{< tab "預知" >}}
<h3 style="color: var(--role-color-wizard);">預知</h3>
{{< block >}}
〈感知魔法〉 — 〈天籟廣播〉 — 〈占卜〉 — 〈真視術〉 — 〈探尋術〉
{{< /block >}}

{{< ability-block title="感知魔法" type="wizard" >}}
{{< cost-tag number=0 >}}每當你接近強大的魔法源時，你就會感覺到一股輕微的刺痛感。你能察覺到該區域有魔法存在，但不清楚它的位置或性質。<p>
{{< cost-tag number=1 >}}你辨別出存在於鄰近範圍內的魔法的位置和大致性質。舉例來說，導引者可能會揭示出你感知到一把魔法劍、一扇被詛咒的門、或一個幻象。<p>
{{< cost-tag number=2 >}}你研究存在於鄰近範圍內的魔法的準確性質，理解其確切的效果。如果你使用這個能力去研究一個神祕的魔法物品，它的名字和效果將會被揭露。
{{< /ability-block >}}

{{< ability-block title="天籟廣播" type="wizard" >}}
{{< cost-tag number=2 >}}你把一段長度最多２０個字的訊息廣播出去。所有位於１公里內的生物都會清楚聽到這段訊息。這段訊息的聲音相當悅耳，且似乎來自四面八方。在廣播期間，環境背景噪音將會被暫時靜音。
{{< /ability-block >}}

{{< ability-block title="占卜" type="wizard" >}}
{{< cost-tag number=1 >}}每場遊戲聚會一次，你可以預視時間長河的流動，瞥見你在不遠未來中的行動結果。在接下來１小時內，你可以透過重新擲骰來避免失敗。一旦你選擇重骰，你就必須接受新的擲骰結果。

或者，你可以請導引者根據你當前的行動選擇脈絡，揭示一個你很有可能遭遇的情境。你可以向導引者詢問關於你感應到未來景象的問題，但這些問題的答案可能會是晦澀模糊的，就彷彿你在嘗試尋找水彩繪畫的細節一樣。
{{< /ability-block >}}

{{< ability-block title="真視術" type="wizard" >}}
{{< cost-tag number=4 >}}在接下來１小時內，你可以看穿任何魔法欺瞞手段。〈真視術〉會透過許多方式為你揭示這個世界：

- 你看見任何隱形的事物。
- 你知曉某個事物是否為幻象。
- 你可以看見任何被魔法改變事物的真實型態。
{{< /ability-block >}}

{{< ability-block title="尋物術" type="wizard" >}}
{{< cost-tag number=3 >}}<b>尋找</b>. 你閉上雙眼，並嘗試定位一個你曾擁有過的物體的位置。只要該物體仍然與你處在同一個宇宙中，你就能得知它的確切位置。<p>
{{< cost-tag number=3 >}}<b>召回</b>. 在你使用〈尋物術〉定位到一個物體的位置後，你可以立刻花費額外３點 AP 來將它召回。該物體將瞬間從它原本所在的位置上消失，並出現在你的手中。
{{< /ability-block >}}

{{< tab "詭術" >}}
<h3 style="color: var(--role-color-wizard);">詭術</h3>
{{< block >}}
〈心音傳聲〉 — 〈舉重若輕〉 — 〈縮放術〉 — 〈回溯〉
{{< /block >}}

{{< ability-block title="心音傳聲" type="wizard" >}}
{{< cost-tag number=1 >}}在接下來１分鐘內，你可以無聲地對鄰近距離內的一個生物用心靈傳話。你的嘴唇會如常動作，就彷彿你在大聲說話一樣，但不會發出任何聲音。在此同時，這個法術的目標會在他的腦海中聽到你的聲音。該生物能夠辨識出你的聲音是直接出現在他腦袋中的。
{{< /ability-block >}}

{{< ability-block title="舉重若輕" type="wizard" >}}
{{< cost-tag number=2 >}}你將超自然怪力注入你的手指。在接下來１分鐘內，你可以毫不費力地舉起並移動任何重量的物體，只要你能合理地應對該物體的尺寸和形狀即可。舉例來說，你可以捏著一個惡霸的衣領並把他舉到空中，或你可以輕鬆自在地移動一個巨人尺寸的鐵砧。
{{< /ability-block >}}

{{< ability-block title="縮放術" type="wizard" >}}
{{< cost-tag number=2 >}}選擇鄰近距離內的一個大小介於螞蟻和馬之間的生物或物體。你把它瞬間放大或縮小。它可以長到比原本大兩倍，或者縮小為原本大小的一半。目標的重量會相應地增加或減少，但它的生命值並不會因此改變。你無法讓目標成長到超過它所在空間能容納的程度。<p>
{{< cost-tag number=4 >}}你可以讓一個大小介於螞蟻和大象之間的生物或物體放大或縮小。
{{< /ability-block >}}

{{< ability-block title="回溯" type="wizard" >}}
你說出力量之語，倒轉鄰近距離內一個物體的命運。這個法術無法被用來影響活物。

你回溯的事物必須要是另一個生物費功夫導致的結果。舉例來說，你可以解開一個生物的鞋帶、解開衣物的縫線、使一件家具散架、或將石雕變回一整塊大理石。

你在每次使用這個法術時選擇你要回溯多少程度的功夫。

{{< cost-tag number=1 >}}最多１分鐘程度的功夫。<p>
{{< cost-tag number=2 >}}最多１天程度的功夫。<p>
{{< cost-tag number=3 >}}最多１個禮拜程度的功夫。<p>
{{< cost-tag number=4 >}}任意程度的功夫。

{{< ability-block title="裂隙門扉" type="wizard" >}}
{{< cost-tag number=5 >}}你召喚出一道壯麗的門扉，開啟一個通往你所在宇宙位於裂隙星界節點的傳送門。（裂隙星界是宇宙諸界中的一個特殊地點，你可以在「宇宙諸界」章節中讀到更多信息。）每個宇宙的裂隙門扉都是獨一無二的；它可能會是個石製的拱門、鐵質大門、一扇樸素的木門、或其他形式。由導引者決定這扇門扉的樣子。

維持這到傳送門是個極其複雜且困難的工作，且它只能維持開啟狀態長達１分鐘。一旦被開啟，任何體型足以穿過傳送門的生物或物體都可以穿過它。

儘管你可以在任何地方召喚出這道門扉，但它在裂隙星界的出口位置仍舊會維持在相同的地方。

{{< ability-block title="超凡意識" type="wizard" >}}
{{< cost-tag number=5 >}}你啟程踏上驚人探索之旅，發掘藏在千萬生物腦中的豐富知識。在接下來一分鐘內，你將進入一種出神狀態，同時你的意識會不斷向外延展，觸碰所有距離你１０００公里內的智慧生物。

這個考驗非比尋常。你對時間與自我的認知將會瞬間超出你正常理解的尺度。你將同時接收到其他生物所體驗的一切──其個數甚至可能高達數十億。

你對知識的追尋必須被限制在一個你能透過進入其他人的視角來合理得知的主題上。舉例來說，你可以學習關於某個物種的真實歷史、得知某個權力最大的存在的身分和位置、發掘某個秘密物品的隱藏地點、或大致熟悉某個文化的信仰和習俗。你無法利用這個法術得到一個人無法合理擁有的知識（像是追求「這個世界上一切的知識」）。導引者會決定你的追求是否合理。

注意。進入其他生物的心靈潛藏著極高的風險。

你可以會因此遭遇難以預料的後果。
{{< /ability-block >}}

{{< ability-block title="創造位面" type="wizard" >}}
你召喚一個陰影位面。它的內容物全部都是幻象，且無法被帶離這個位面。你可以在這個位面內創造出任何你能想像的事物，除了智能生物之外。

儘管這個陰影位面內的事物不管看起來或感覺起來都跟真的一樣，但它們仍僅僅是幻影。生物不會被你在這個位面中創造的事物所傷害。

{{< cost-tag number=2 >}}你召喚出一個規模不超過一棟豪宅或城堡的小型位面。<p>
{{< cost-tag number=4 >}}你召喚出一個規模不超過村莊大小的位面。<p>
{{< cost-tag number=6 >}}你召喚出一個沒有規模限制的位面。<p>
{{< cost-tag number=8 >}}你召喚出一個沒有規模限制、也沒有安全限制的位面。你在這個位面中創造的事物現在將能夠傷害其他生物。（你也包括在內。）
{{< /ability-block >}}

{{< ability-block title="造物術" type="wizard" >}}
{{< cost-tag number=5 >}}你輕觸原始魔法，將其能量轉變成一個非魔法、無生命、且大小足以裝進一個５公尺立方的立方體空間內的物體。你創造出的物體完美無缺，且在各個方面都無比真實。你可以創造出任何能夠容納於這個空間內的物體，包括珍貴的寶石、毫無瑕疵的武器、或其他物體的完美複製品。

這個法術要求完美的表現，且會對你作為法師的能力帶來極大的負擔。在你的道具欄上記錄你施放了這個法術的次數。你每次使用這個法術都會使它的花費額外增加１點 AP。
