title: "醫生"

醫生輕柔地觸摸同伴的手臂，修復他的身體和靈魂。他俯身檢視死者，探究是什麼造成其殘酷的命運。他能感知逝去的亡者，並與迷途的靈魂交談。

如果你想成為隊伍中的治療者、或者想要玩弄生與死的力量，醫生會是很好的選擇。

{{< tab "治癒" >}}
<h3 style="color: var(--role-color-doctor);">治癒</h3>
{{< block >}}
〈癒合傷勢〉 — 〈舒緩之語〉 — 〈治療術〉 — 〈復原術〉
{{< /block >}}

{{< ability-block title="癒合傷勢" type="doctor" >}}
{{< cost-tag number=1 >}}你輕柔地觸碰一個生物，立即讓他恢復５點生命值，並使輕微的傷口癒合，像是割傷和瘀傷。這個法術無法治療身體殘疾、永久性創傷、或治癒疾病。你無法在戰鬥中使用這個法術。
{{< /ability-block >}}

{{< ability-block title="舒緩之語" type="doctor" >}}
{{< cost-tag number=1 >}}你對鄰近距離內一個生物說出安慰人心的話語，有效緩解他的焦慮、痛苦和不適。這個效果會持續一個小時。在這期間，該生物不會受到恐懼或混亂的影響。且在短時間內，他的感覺將前所未有的良好。
{{< /ability-block >}}

{{< ability-block title="治療術" type="doctor" >}}
{{< cost-tag number=2 >}}你用關懷的觸碰去擁抱一個生物，恢復他所有的生命值。在一小段時間之後，該生物受到的短期損傷將被治療，這個法術無法治療永久性的負面狀態。如果你對隊伍的另一個成員使用〈治療術〉，則他可以貢獻自己任意數量的 AP 來分攤這個法術的花費。
{{< /ability-block >}}

{{< ability-block title="復原術" type="doctor" >}}
{{< cost-tag number=7 >}}你觸碰一個活著的生物，在幾分鐘內將其完全恢復到他的正常狀態。（你無法在戰鬥中使用這個法術。）這個法術會恢復所有的生命值，並治療任何的殘疾或損傷。如果你對隊伍的另一個成員使用〈復原術〉，則他可以貢獻自己任意數量的 AP 來分攤這個法術的花費。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "改造" >}}
<h3 style="color: var(--role-color-doctor);">改造</h3>
{{< block >}}
〈睡眠術〉 — 〈硬化表皮〉 — 〈假死術〉 — 〈整容術〉 — 〈生命連結〉
{{< /block >}}

{{< ability-block title="睡眠術" type="doctor" >}}
{{< cost-tag number=2 >}}你輕唱一首簡短的搖籃曲，讓鄰近範圍內任意數量的平民陷入沉睡長達最多一個小時的時間。你可以撰寫你自己的搖籃曲，也可以借用其他作者的作品。在遊戲桌上將其唱出來。

受到這個法術影響的生物會頹然倒下並進入夢鄉，但他會在受到傷害時立即清醒。

 {{< cost-tag number=x >}}你可以增強這個法術，使其能夠影響鄰近範圍的爪牙。你每要讓一個爪牙睡著，就得額外花費１點 AP。
{{< /ability-block >}}

{{< ability-block title="硬化表皮" type="doctor" >}}
{{< cost-tag number=1 >}}你觸碰一個生物，使他的體表腫脹並硬化。在這段時間內，他將對外在痛楚感到麻木。他硬化的體表厚皮能夠吸收最多２點傷害的物理打擊。任何會造成超過２點傷害的攻擊都將正常擊中該生物。該生物的皮膚會在１分鐘後恢復正常。
{{< /ability-block >}}

{{< ability-block title="假死術" type="doctor" >}}
{{< cost-tag number=3 >}}你使鄰近範圍內的最多６個生物陷入假死狀態，持續最多一個小時。（你只能對自己或自願的盟友施放這個法術。）處於假死狀態的生物無法視物、感覺、說話、或移動，但會維持半清醒的意識，並能夠聽見他們周圍發生的事情。

受到這個法術影響的生物摸起來冰冷，探測不到脈搏，從旁觀者看來就像是死了一樣。只有魔法能夠揭露這個假象。受到影響的生物會在受到傷害時立刻清醒，並可以自主選擇在任何時刻恢復過來。
{{< /ability-block >}}

{{< ability-block title="整容術" type="doctor" >}}
{{< cost-tag number=4 >}}你觸碰一個生物，重塑他的面部外觀。你可以讓他看起來變得更老、更年輕、或變得像另一個完全不同的人物。如果你想讓他看起來像是某個特定的人物，則你在施放這個法術時必須要看著目標長相的參照圖。

這個變形是永久的，且只能透過〈復原術〉或再次施放〈整容術〉來恢復。
{{< /ability-block >}}

{{< ability-block title="生命連結" type="doctor" >}}
{{< cost-tag number=4 >}}你讓鄰近距離內的二個生物彼此的生命能量纏結在一起。你在每次使用這個能力時選擇一個效果：

- **雙向連結**. 當被連結生物中的任何一方受到傷害時，另一個生物也會受到同樣的傷害。這個效果不會作用於同時擊中這兩個生物的傷害（比如說範圍法術）。
- **單向連結**. 選擇其中一個被連結生物。對該生物受到傷害時，另一個生物也會受到同樣的傷害。
- **能量導線**. 當被連結生物中任何一方被治療時，另一個生物也會得到同樣的治療。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "死靈" >}}
<h3 style="color: var(--role-color-doctor);">死靈</h3>
{{< block >}}
〈死亡感應〉 — 〈死者交談術〉 — 〈活化屍體〉 — 〈附身亡魂〉
{{< /block >}}

{{< ability-block title="死亡感應" type="doctor" >}}
{{< cost-tag number=0 >}}你能自然地能感應到鄰近範圍內是否存在死者的殘餘物（也就是他們的靈體），但無法感應到它的所在位置。導引者會在這個感應被觸發時告知你。<p>
{{< cost-tag number=1 >}}你能感應到鄰近範圍內是否有任何死亡的智慧生物，以及他死了多長的時間。你能感應到這些生物死亡的地點，以及關於他死亡的大致性質；你能感應到他的死亡是出於自然因素、意外事故、還是謀殺。<p>
{{< cost-tag number=1 >}}你擴展你的視覺，讓你能夠看見隱形的靈體生物。這個效果會持續直到你離開該區域。
{{< /ability-block >}}

{{< ability-block title="死者交談術" type="doctor" >}}
{{< cost-tag number=1 >}}如果你察覺到鄰近範圍內有一個或多個靈體生物（比如說鬼魂），則你可以和他們交談。你不需要看見這些靈體，只要能感應到他們的存在即可，比如說透過〈死亡感應〉能力。

你可以與靈體進行最長一分鐘的對話，並透過心靈感應與其交流。你們之間必須要有至少一種共通語言才有辦法互相理解。該靈體必須要有回應的意願才能使對話成立，如果他拒絕與你交談，則你花費的 AP 點會被退回。
{{< /ability-block >}}

{{< ability-block title="活化屍體" type="doctor" >}}
{{< cost-tag number=1 >}}你活化鄰近距離內的一具屍體或骨骸，使其成為你的僕從。它能夠四處移動，但無法離開你所在的場景。這個法術會在你離開該場景時結束。

你能透過心靈感應控制這個僕從。它沒有自我意識，且只會聽從你的命令。這個生物擁有３點生命值，且可以進行能造成１點傷害的攻擊。
{{< /ability-block >}}

{{< ability-block title="附身亡魂" type="doctor" >}}
你觸碰一個活著的生物，使他成為一個鄰近距離內靈體的宿主。選擇一項：
{{< cost-tag number=5 >}}黑暗住客. 你把該靈體融入目標生物的體內，這個靈體生物將永遠寄宿在這位宿主的心靈中。這個靈體無法控制宿主的身體，但能體驗其感受，並透過心靈感應與宿主對話。<p>
{{< cost-tag number=7 >}}竊取肉身. 宿主的意識將消散不見，而靈體將取而代之，並能夠完全控制宿主的身體。宿主的肉體仍會保留一部份的記憶殘片，並會偶爾浮現出來。（如果要對頭目使用這個能力，則他的剩餘生命值必須要低於其最大生命值的２０％）
{{< /ability-block >}}
{{< /tab >}}

{{< tab "傷害" >}}
<h3 style="color: var(--role-color-doctor);">傷害</h3>
{{< block >}}
〈潰傷術〉 — 〈化合刺激〉 — 〈苦痛折磨〉 — 〈凋零術〉 — 〈心臟咒殺〉
{{< /block >}}

{{< ability-block title="潰傷術" type="doctor" >}}
{{< cost-tag number=0 >}}你抓住觸及距離內的一個有機體生物，製造出一道持續壞死的傷口。這個傷口的肌肉組織會變成黑綠色，並從你觸碰的地方蔓延開來。這個法術會在擊中時立即造成１點傷害，並在目標的下個回合開始時再次造成傷害。（總共造成２點傷害。）受影響的生物只能透過〈復原術〉法術來恢復這些失去的生命值。
{{< /ability-block >}}

{{< ability-block title="化合刺激" type="doctor" >}}
{{< cost-tag number=1 >}}你觸碰一個生物，讓他在接下來一小時內獲得一組綜合效應。你可以從列表中選擇最多三個效果：

清醒不眠、警惕、增強自信、亢奮、發笑、鮮明感官、放鬆、發汗、噁心反胃、寒顫、煩躁易怒。
{{< /ability-block >}}

{{< ability-block title="惡病纏身" type="doctor" >}}
{{< cost-tag number=3 >}}你觸碰一個生物，對他施加一個惡病詛咒，持續最多１個月。你在每次使用這個能力時選擇一個病症：

- **愚蠢心智**. 該生物會完全背離原本的性格，以憨傻且無憂無慮的方式行動。這可能會改變他的行動目標。
- **笨手笨腳**. 該生物的四肢變得笨拙難用，他無法有效地使用武器，並會不斷把東西弄掉在地上。
- **萎靡不振**. 該生物無法恢復生命值。

{{< rolling-tag >}}

如果你骰出失敗的結果，則這個惡病會獲得傳染性，快速傳播到任何與目標接觸的生物身上。<br/>
如果你骰出災難大失敗的結果，則你也會被感染。
{{< /ability-block >}}

{{< ability-block title="衰老術" type="doctor" >}}
{{< cost-tag number=4 >}}你觸碰一個生物，使他的肉體凋零老化。他的壽命不會改變，但他會體驗到年紀衰老的影響。他將開始感覺死亡逐漸迫近——而這很可能會使該生物的行為和目標產生改變。
{{< /ability-block >}}

{{< ability-block title="心臟咒殺" type="doctor" >}}
{{< cost-tag number=6 >}}你觸碰一個平民或爪牙，對他的心臟施加詛咒。（這個法術只能作用於擁有心臟的生物。）你可以選擇立即讓他的心臟爆炸，將其殺死。或者，你可以把他的心臟變成一顆定時炸彈，將心臟爆炸的時間延後，最久延後到一年後。

如果你對頭目使用這個法術，則這會變成一個觸發效果。如如果該頭目在詛咒期間讓生命值降到了最大生命值的２０％或以下，則他的心臟會爆炸並立即死亡。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "感覺" >}}
<h3 style="color: var(--role-color-doctor);">感覺</h3>
{{< block >}}
〈知覺調幅〉 — 〈操弄感官〉 — 〈認知錯覺〉 — 〈遺忘術〉
{{< /block >}}

{{< ability-block title="知覺調幅" type="doctor" >}}
{{< cost-tag number=1 >}}你改變鄰近距離內一個生物某個生理感官的性質和強度，持續１小時。選擇一個效果：

- **溫度**. 該生物會感到一股讓人放鬆下來的內在暖意或外在涼意。或者，你可以讓他感覺太熱或太冷。
- **食慾**. 該生物會覺得食物和飲料令他作嘔、或湧現出一股無法滿足的飢餓和渴意、或者感到吃飽喝足。
- **舒適**. 該生物會感覺他在任何地方或坐或躺都能放鬆自在。或者，無論他怎麼擺佈身體都不自在。
{{< /ability-block >}}

{{< ability-block title="操弄感官" type="doctor" >}}
{{< cost-tag number=1 >}}你觸碰一個生物，並操弄他對顏色的感受能力。舉例來說，你可以讓一個目標生物把綠色看成紫色。或者你也可以組合複數的顏色，讓目標把黃色和綠色都看成藍色。你也能讓目標看到的顏色全變成過飽和色、不飽和色，或讓他變成單色視覺。你可以選擇任意數量的改變和組合，只要它們的效果不會彼此重疊。<p>
{{< cost-tag number=2 >}}你觸碰一個平民或爪牙，在接下來這一天內把他的其中一種感官轉移到宇宙中的某個隨機位置。你可以選擇並傳送下列其中一種感官：視覺、觸覺、聽覺、味覺、嗅覺。該生物會體驗到那個感官在新環境的感受，就彷彿他真的在那裡一樣。
{{< /ability-block >}}

{{< ability-block title="認知錯覺" type="doctor" >}}
{{< cost-tag number=3 >}}你觸碰一個生物，使他對特定的事物或類別產生錯誤的認知。舉例來說，你可以讓他把鎮長看成小丑，或把所有的狼都看成小狗。或者，你可能會讓他覺得新鮮出爐的麵包聞起來像火一樣。這個法術會持續直到你取消它的效果、或直到該生物因為這個錯覺的效果而被傷害為止。
{{< /ability-block >}}

{{< ability-block title="遺忘術" type="doctor" >}}
{{< cost-tag number=4 >}}你觸碰一個生物，抹除他從今天算起最多１個月內的記憶。這個能力只會作用於沉睡中的生物。<p>
{{< cost-tag number=x >}}你可以透過花費額外的 AP，來增加記憶消除的時間長度。每１點 AP 都會增加最多額外一年的效果。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "鑑識" >}}
<h3 style="color: var(--role-color-doctor);">鑑識</h3>
{{< block >}}
〈遺體鑑識〉 — 〈診斷〉 — 〈怪奇案件〉
{{< /block >}}

{{< ability-block title="遺體鑑識" type="doctor" >}}
{{< cost-tag number=1 >}}你對一個觸及距離內的遺體進行評估。這具遺體的狀況必須要完好到足以讓你檢視它。（它不能完全被摧毀。）在一分鐘的檢查之後，你能夠確定該生物的確切死因、死亡時間和日期。
{{< /ability-block >}}

{{< ability-block title="診斷" type="doctor" >}}
{{< cost-tag number=0 >}}你觸碰一個生物，延展你的感官來診斷他身上的神秘病症，像是疾病、法術、或詛咒。你必須正確地猜出這個病症的全名。如果你猜錯，則這個法術會失敗，且你必須花費１點 AP 才能再試一次。

導引者會給你好幾個空格，表示這個病症的全名是由多少單詞組成，每個單詞又是由幾個字母組成。你可以從字母開始猜，看看這個字母是否包含病症全名之中。如果你猜對了，那麼導引者會揭示這個字母在全名中出現的所有位置。如果你總共猜錯６次，則這個法術失敗。

{{< rolling-tag >}}

||
| :-- |
| 20: 導引者會先填上病症全名中一半數量的字母。 |
| 11-19: 導引者會先填上一個母音和二個子音。 |
| 1-10 導引者會先填上一個子音。 |

一旦你診斷出這個病症，那麼你以後將在觸碰到其他生物時自動辨識出它的存在。
{{< /ability-block >}}

{{< ability-block title="怪奇案件" type="doctor" >}}
{{< cost-tag number=3 >}}如果場景中存在著明顯可疑的事物，像是屍體或是牆上的血跡，則你可以透過回想熟悉的事物進行調查。你必須在遊戲桌上侃侃而談你過去的經驗，說明你熟悉哪些事物。

選擇三個你已經知道的場景細節。若你需要更多的細節，詢問導引者你發現到了什麼。

你可以接著開始你的短講，說一些像是：「這讓我想起關於 __________ 的怪奇案件。」來解釋你之前在場景中看到的細節。

在你說完之後，導引者會為你揭示一個與場景有關的有用線索。導引者會從下列這些線索中選擇一項：
- 某個涉及該場景的犯罪者的動機。
- 附近的一個隱藏訊息、符號、或其他能揭示出某個涉及該場景的人物的名稱或隸屬的線索。
- 某個涉及該場景的人物接下來可能會去的地方。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "傳奇能力"  >}}
<h3 style="color: var(--role-color-doctor);">傳奇能力</h3>

{{< ability-block title="起源之令" type="doctor" >}}
{{< cost-tag number=6 >}}你觸碰一個生物，並決定要逆轉或加速他生命階段的發展。（如果要對頭目使用這個能力，則他的剩餘生命值必須低於其最大生命值的２０％。）舉例來說，你可以將一個成年生物逆轉回孩童，給予他再次成長變老的機會。

{{< ability-block title="永恆沉眠" type="doctor" >}}
{{< cost-tag number=4 >}}你觸碰一個觸及距離內的生物，使他陷入永恆寧靜的沉睡之中。

你可以選擇為這個法術設定一個會使其結束的觸發條件；舉例來說，詛咒他直到被某個家族成員給擁抱之前都永遠沉睡、或者必須要破壞某個特定的物體才能終止這個法術的效果。

你可以在任何時候選擇終止這個法術。且你同時只能對一個生物使用這個法術的效果；如果你再次施放它，則前一個法術將會終止，讓前一個法術影響的生物從沉睡中甦醒。

（如果要對頭目使用這個能力，則他的剩餘生命值必須低於其最大生命值的２０％。）

{{< ability-block title="苦澀的贈禮" type="doctor" >}}
{{< cost-tag number=9 >}}你觸碰一個沒有自我意識的生物，並賦予牠一個名為「自我」的苦澀贈禮。你必須在遊戲桌上朗誦一段關於存在意義的句子。這可以是書中的一段文字、一首詩曲、或其他陳述著生命意義的內容。

這個法術的目標將會在接下來一分鐘內失去意識，同時牠的身體將會急速經歷一百萬年的自我意識進化。這個進化過程會以你意想不到的方式改變這個動物的身體特徵和能力。

這個重獲新生的生物在來到這個世界時，將會得到運用一種語言的能力、對自我認知的概念、以及對牠過往生活的模糊認知，這包括了一些重大時刻的記憶。換句話說：牠現在是個和你一樣的「人」了。

該生物會知道是你賦予了牠生命。
