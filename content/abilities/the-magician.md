魔術師輕揮手臂，變出一大群動物幻影來取悅觀眾。他催眠對手，讓他僵在原地。他凝視他人的雙眼，進入對方的夢境，自由形塑其認知。

如果你想要運用操控心靈和感官的法術，或想扮演一個令人目眩神迷的表演者和狡猾的操弄騙徒，那麼魔術師會是一個很好的選擇。

{{< tab "誤導" >}}
<h3 style="color: var(--role-color-magician);">誤導</h3>
{{< block >}}
〈魔術把戲〉 — 〈催眠〉 — 〈轉移怒火〉 — 〈海市蜃樓〉 — 〈隱形術〉
{{< /block >}}

{{< ability-block title="魔術把戲" type="magician" >}}
{{< cost-tag number=0 >}}你創造出一個微弱的魔法效果來給周圍的人們帶來驚奇、歡樂或困惑。<br/>
你在每次使用這個能力時從下列效果中選擇任意組合：

- **光源**. 你創造出無害的光源，例如搖曳火焰或構成圖案的火花。你也可以熄滅或點燃小型光源，像是火把。
- **聲音**. 你創造出一個微小而簡短的聲音效果，例如風鈴聲、觀眾的掌聲、或某人的低語。
- **味道**. 你創造出一股任意種類的微弱氣味，例如新鮮出爐的餡餅或污水池的味道。
- **觸感**. 你讓鄰近範圍內一個或多個生物感受到一股溫和的體感，例如有人輕拍他的肩膀、一陣涼風、或一陣雞皮疙瘩。
{{< /ability-block >}}

{{< ability-block title="催眠" type="magician" >}}
{{< cost-tag number=1 >}}你透過視覺幻象來迷惑鄰近距離內的一個平民或爪牙。（該生物必須要能視物。）直到你離開該場景前，該生物將無法移動、採取行動，或對談話做出反應。這個法術會在該生物受到傷害時結束。
{{< /ability-block >}}

{{< ability-block title="轉移怒火" type="magician" >}}
{{< cost-tag number=1 >}}你做出一個指示性的手勢，轉移鄰近距離內一個對你有敵意的生物的注意力。目標的怒火將會被轉移到他的另一個對手身上。（如果目標在鄰近範圍內沒有其他敵人存在，則這個法術不會起作用。）這個效果會在你傷害該生物時被終止。
{{< /ability-block >}}

{{< ability-block title="海市蜃樓" type="magician" >}}
{{< cost-tag number=3 >}}你創造出一個長寬高都不超過１００公尺的大型幻象。這個幻象可以是任何足以容納於該空間的事物，且你能夠設定它表現出固定的影像循環。舉例來說，你可以創造出一座綠洲、一堆寶藏、或一個保衛自己領土的笨重巨人的幻象。

如果某個生物接觸或穿過了這個幻像，那他就不會再看到這個幻象。擁有探測魔法能力的生物可以察覺到這個幻象是虛假的。
{{< /ability-block >}}

{{< ability-block title="隱形術" type="magician" >}}
{{< cost-tag number=3 >}}你讓鄰近距離內的一個生物或物體在接下來一分鐘內隱形。如果目標是一個生物，則他穿戴和攜帶的一切也都會跟著隱形。如果該生物在隱形狀態下進行攻擊，則他將重新現形。<p>
{{< cost-tag number=x >}}你每額外指定一個生物或物體做為這個法術的目標，就需要多花費１點 AP。
{{< /ability-block >}}

{{< tab "幻體" >}}
<h3 style="color: var(--role-color-magician);">幻體</h3>
{{< block >}}
〈分身鏡影〉 — 〈忌厭魅影〉 — 〈活物幻體〉
{{< /block >}}

{{< ability-block title="分身鏡影" type="magician" >}}
{{< cost-tag number=1 >}}你瞬間消失，並伴隨著兩個幻象分身重新出現。你的分身會跟著你一起移動，並完美模仿你的所有動作。

如果有其他生物試圖攻擊你，則導引者必須擲一枚硬幣。如果擲到「正面」，則你成為攻擊目標；如果擲到「反面」，則改為由你的其中一個分身成為攻擊目標。一旦你的分身受到傷害，它就會消失不見。
{{< /ability-block >}}

{{< ability-block title="忌厭魅影" type="magician" >}}
{{< cost-tag number=2 >}}你讀取鄰近距離內一個 NPC 的心靈，創造出一個會讓對方感到極度厭惡的生物幻影。只有你和這個法術的目標能夠看見和聽見這個幻影。

在接下來１０分鐘內，這個幻影會無情地嘲弄和挑釁你的目標，吸引他全部的注意力。

你可以直接控制這個幻影，讓它操弄或使用小型物體。舉例來說，你可以讓它偷走目標身上的一件物品，引導目標去追趕幻影。或者，你可以放任幻影自由發揮，讓導引者描述它的行為舉止。
{{< /ability-block >}}

{{< ability-block title="活物幻體" type="magician" >}}
{{< cost-tag number=3 >}}想像一個生物的樣子。你創造出該生物的逼真幻體，讓它出現在鄰近範圍內的某處。它的外觀、動作、和聲音都跟你想像的生物如出一轍，就連摸起來的觸感都很真實。這個幻體具有６點生命值，並會在生命值歸零時消失不見。

這個幻體可以獨立行動，並能夠離開你的身邊。你可以設定這個幻體的行為模式。舉例來說，你可以讓它打掃一間房子、在一個區域巡邏、裝忙、或給它設定一組任務和行動的組合。你也可以為它設定規則，像是「無論如何都不能傷害任何人」或者「不要讓任何人通過這扇門」。

當這個幻體在你的鄰近範圍內時，你跟它共享一道心靈感應的連結，且可以透過這道連結在你的回合直接控制它的行動。你可以讓它四處移動、採取行動、或替你說話。它可以持握並使用武器來進行基本攻擊。

在１天之後，這個幻體會自動避開其他生物的視線，接著消失不見。

{{< cost-tag number=4 >}}如果你想要用這個法術來創造某個特定生物的幻體，則它改為需要花費４點 AP。你創造出一個該生物的完美複製體，無論看起來和聽起來都跟他一模一樣。這個複製幻體能夠騙過除了該生物最親密的朋友和家人之外的所有人。
{{< /ability-block >}}

{{< tab "透識" >}}
<h3 style="color: var(--role-color-magician);">透識</h3>
{{< block >}}
〈祕法魔眼〉 — 〈心靈低語〉 — 〈傳訊術〉 — 〈通譯術〉 — 〈洞察術〉
{{< /block >}}

{{< ability-block title="祕法魔眼" type="magician" >}}
{{< cost-tag number=1 >}}你暫時獲得能夠看穿自然現實的能力。在接下來一小時內，你能看見以下事物：

- **魔法**. 任何當前正受到魔法影響的人或物體，周圍都會環繞著淡淡的靈光。
- **幻象**. 任何生物或物體的幻象都會輕微地搖曳閃動，但你無法看穿它的真實形態。
{{< /ability-block >}}

{{< ability-block title="心靈低語" type="magician" >}}
{{< cost-tag number=0 >}}你透過心靈感應向鄰近距離內的一個生物傳送一段訊息。他會知道是你在他的心中說話，且他會聽到你的聲音，就彷彿你正在大聲說話一樣。<p>
{{< cost-tag number=1 >}}你創造出一道與該生物的連結，讓他能夠回應你的心靈低語。如果他回應了你，那你將可以在接下來一分鐘內和他透過心靈感應溝通。
{{< /ability-block >}}

{{< ability-block title="傳訊術" type="magician" >}}
{{< cost-tag number=1 >}}選擇一個你曾經見過的生物。你必須要知道它當前所在的大致位置。你們可以透過心靈感應彼此溝通，長達最多１分鐘的時間。<p>
{{< cost-tag number=4 >}}你創造一道與該生物的心靈連結，即使你不知道他的所在位置也一樣。該生物可以在任何地方。
{{< /ability-block >}}

{{< ability-block title="通譯術" type="magician" >}}
{{< cost-tag number=1 >}}你進入鄰近距離內一個生物的心靈，並在接下來１０分鐘內暫時學會他的語言。

當你施放這個法術時，你還可以透過玩一場比手畫腳遊戲，嘗試永久學會該生物的語言。（這場遊戲會跳脫角色扮演，並在遊戲桌上進行。）導引者會在紙上寫下一個單詞或短句，然後把它秘密地遞給你。

設定一個計時器，你有１分鐘的時間安靜地透過比手畫腳的方式表達這個單詞或短句。如果其他玩家正確地猜出這個單詞或短句，則你將成功學會該語言。
{{< /ability-block >}}

{{< ability-block title="洞察術" type="magician" >}}
{{< cost-tag number=3 >}}你凝視鄰近距離內一個生物的心靈，發掘對方的主要意圖。你在每次施放這個法術時選擇一個效果：

- 你得知該生物在接下來１０分鐘內打算去做什麼事情。
- 你得知該生物在長期未來中打算去完成什麼目標。
- 你得知該生物打算對你所選擇的一個特定生物做什麼事情。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "造物" >}}
<h3 style="color: var(--role-color-magician);">造物</h3>
{{< block >}}
〈哨戒小鳥〉 — 〈海倫的夢幻盛宴〉 — 〈天眼〉 — 〈影界安居〉
{{< /block >}}

{{< ability-block title="哨戒小鳥" type="magician" >}}
{{< cost-tag number=1 >}}你在鄰近距離內的某處召喚出一隻幻影鳥，讓它監看一塊視線範圍內的區域。它會持續待在那裡直到你將其解消。你同時只能召喚一隻哨戒小鳥。

你和哨戒小鳥之間共享一道心靈連結。它會在目擊你指定的事物時透過心靈傳訊通知你，像是某個特定生物進入了房間、或是門被打開。如果你正在睡覺，則哨戒小鳥會把你叫醒，告訴你它看見了什麼。
{{< /ability-block >}}

{{< ability-block title="海倫的夢幻盛宴" type="magician" >}}
{{< cost-tag number=4 >}}想像一頓使用任意烹飪風格精心製作的大餐饗宴。你召喚出你想像中的食物、飲料、以及你所選擇的傢俱（若有需要）、餐具、和餐桌裝飾。你可以召喚出足夠餵飽最多２０人的食物。享用這頓大餐能讓每個人恢復４點生命值。
{{< /ability-block >}}

{{< ability-block title="天眼" type="magician" >}}
{{< cost-tag number=2 >}}你召喚一個微小的幻影僕從，它會懸浮在你的肩膀上，並在這場遊戲聚會的剩餘時間內一直跟在你身邊。「天眼」看起來就像一朵盛開的花朵，且有一隻位於中央位置的眼睛。它是隱形的，除了你指定的生物之外，其他人都看不見它。你可以把「天眼」部署到你視線之外的區域。舉例來說，你可以派它繞過轉角進入另一個房間進行搜索。

「天眼」能告訴你它所處的空間內存在著多少個生物，但無法告訴你其種類。它也會透過心靈傳送，給你一份它經過空間的簡陋地圖。舉例來說，如果你把它派到隔壁的房間，那麼你會知道那個房間的概略大小、形狀、以及任何主要的結構特徵，像是柱子或坑洞。
{{< /ability-block >}}

{{< ability-block title="影界安居" type="magician" >}}
{{< cost-tag number=5 >}}你把手放在現世位面中的一棟住宅上，你立即將其內部結構複製到陰影位面之中。這份複製品無論看起來和摸起來都相當真實，但它僅僅是個幻象；你無法將這個幻象的一部分從陰影位面中帶出來。

這棟住宅的大小可以介於棚子到莊園或甚至城堡之間。這棟位於現世位面的建築物的物理入口同時也將變成通往「影界安居」的秘密通道。你可以在任何時候賦予或撤銷任何人進入的權限。知道「影界安居」的存在且擁有進入權限的生物，可以在開門時選擇開啟通往「影界安居」的傳送門。除此之外的情況，這扇門會如常作為現世位面住處的入口而被使用。

「影界安居」會永久存在，直到它被外力驅散、你將其解消、或這個法術被再次施放。當法術被終止時，任何待在裏頭的生物都會被立刻趕出來，透過傳送門被丟回現世位面。
{{< /ability-block >}}
{{< /tab >}}

{{< tab "心控" >}}
<h3 style="color: var(--role-color-magician);">心控</h3>
{{< block >}}
〈迷惑術〉 — 〈蕾恩的歡樂美夢〉 — 〈歡快泛濫〉 — 〈恐懼術〉
{{< /block >}}

{{< ability-block title="迷惑術" type="magician" >}}
{{< cost-tag number=1 >}}你擾亂鄰近距離內一個生物的心靈，讓他對某個特定事物感到略微迷惑，持續直到你離開該區域。

舉例來說，你可以讓商人對他商品的價值感到困惑、讓警衛迷惑於究竟誰有權通過、或者讓狗狗懷疑牠自己到底是不是個乖孩子。如果你欺騙得太過頭，比如說試著拿石頭交換一件無價之寶，那麼這股混亂迷惑的效果就會被解除。
{{< /ability-block >}}

{{< ability-block title="蕾恩的歡樂美夢" type="magician" >}}
{{< cost-tag number=1 >}}你對鄰近距離內的一個生物低語並揮手。該生物在下次入睡時，將經歷一段能深深安撫他心靈的夢境。他在醒來時將感覺被愛，並忘掉所有怨恨，這個效果會持續一整個禮拜的時間。
{{< /ability-block >}}

{{< ability-block title="歡快氾濫" type="magician" >}}
{{< cost-tag number=1 >}}你詛咒鄰近距離內的一個生物，持續１０分鐘。這個詛咒會在該生物受到傷害時被解除。選擇一項：

- **大笑**. 他捧腹大笑，無法移動。他聽到的所有聲音對他來說都顯得滑稽無比。
- **跳舞**. 他會聽見自己最喜歡的曲調並開始跳起舞來，就彷彿他不跳舞會死一樣。
- **模仿**. 他開始模仿一個你所選擇的生物。例如，你可以讓他表現得像隻家貓。

{{< cost-tag number=4 >}}你詛咒鄰近範圍內任意數量的平民。
{{< /ability-block >}}

{{< tab "騷亂" >}}
<h3 style="color: var(--role-color-magician);">騷亂</h3>
{{< block >}}
〈光耀迸發〉 — 〈迷幻箭〉 — 〈鬆綁術〉 — 〈混亂彈球〉
{{< /block >}}

{{< ability-block title="光耀迸發" type="magician" >}}
{{< cost-tag number=0 >}}你伸出手指，讓一道疾馳的閃耀光束射向你能看見的一個目標。擲骰決定你是否成功命中目標。

你可以讓這道閃耀光束在接觸時炸裂出一道劈啪作響的能量，對擊中的目標造成２點傷害。或者，你可以把這道閃耀光束變成無害的煙火，在空中爆炸，創造出任何形狀和色彩的圖案。
{{< /ability-block >}}

{{< ability-block title="迷幻箭" type="magician" >}}
{{< cost-tag number=4 >}}你召喚出四道閃爍著耀眼強光的稜光箭矢。你可以引導每道稜光箭矢各自射向任何你能看見的目標。

每道稜光箭矢會在擊中時造成２點傷害。如果四道稜光箭矢全都擊中同一個生物，則該生物還會受到你〈催眠〉能力的影響。
{{< /ability-block >}}

{{< ability-block title="鬆綁術" type="magician" >}}
{{< cost-tag number=2 >}}你轉動你的手指，彷彿在解開一團毛線一樣。從鄰近距離內一個你能看見的生物開始，你創造出一道接連蔓延出去的魔法惡作劇效果，鬆開任何鄰近範圍內生物身邊被拴住、綑綁、或緊繫的事物。例如：

- 袋子的束口被鬆開，內容物因此傾瀉而出。
- 鞋帶被鬆開、腰帶被解開、且襯衫的鈕扣被解開。
- 拴住的寵物被釋放。
{{< /ability-block >}}

{{< ability-block title="混亂彈球" type="magician" >}}
{{< cost-tag number=3 >}}你在手中召喚出一顆櫻桃大小的彈力球。當你把這顆球丟出去時，它會在撞到任何它接觸到的事物時高速彈開，並避開所有生物。這顆球每彈跳一次，它就會稍微加速，用越來越快的速度四處彈跳，直到在１分鐘後於空氣中消散。

這顆彈力球在這短短時間內的暴行將粉碎該場景中的陶製品、玻璃、輕型傢俱，和大部分的易碎物品。太難過了！
{{< /ability-block >}}

{{< tab "傳奇能力"  >}}
<h3 style="color: var(--role-color-magician);">傳奇能力</h3>

{{< ability-block title="支配術" type="magician" >}}
{{< cost-tag number=8 >}}你在接下來一小時內支配鄰近距離內一個生物的心靈。該生物將會遵循任何你給予他的口頭命令，且不會有絲毫的反抗或猶豫。當這個法術結束時，他仍會記得這段期間發生的所有事。

這個法術附帶著詛咒。如果你使用〈支配術〉去傷害其他生物，那麼你將永遠喪失你的角色，且他將因此變成一個 NPC 頭目。現在起導引者將會扮演你的角色，讓他成為故事中的反派。

如果你使用這個法術的目的會導致你的角色變成 NPC，則導引者會在你施放法術前告訴你；你必須選擇要同意放棄你的角色，或是選擇放棄施放〈支配術〉去傷害其他生物。
{{< /ability-block >}}

{{< ability-block title="魔幻飛航" type="magician" >}}
{{< cost-tag number=5 >}}你從其他維度召喚出一個不可思議的熱氣球，這個氣球的大小能夠容納你和你的隊伍。你在施放這個法術時必須處於室外，並站在一個足以讓熱氣球成形的空曠空間。

這個熱氣球有３０公尺高，外觀可以是任何你想要的造型。這個熱氣球會由一個小型的幻象生物來駕駛，他能夠跟你和你的乘員們說話，並會遵循指示。（這個生物預設的外觀是一隻戴著護目鏡的小熊貓，但你想要的話可以選擇任何類似體型的生物。）如果你之後再次施放這個法術，這位駕駛員仍然會記得你和你曾經去過的地方。

這個熱氣球的動力是由一團位於車廂上方的燃燒球體所驅動，它能根據你的命令調整其火力。這個熱氣球能上升到與世界最高峰相同的高度，並能以最快時速３０公里的速度飛行。這個熱氣球的車廂具有４０點生命值，熱氣球部分則具有１０點生命值。

你可以隨時讓這個氣球消散；它將會跟駕駛員一起消失不見，回到它原本的家。
{{< /ability-block >}}

{{< ability-block title="完美的禮物" type="magician" >}}
{{< cost-tag number=4 >}}你瞥視鄰近距離內一個生物的內心，找出能觸動他內心的事物。接著，你召喚出一個情感層面上的無價之寶，贈與他當作禮物。

你決定該生物在收下這個禮物時會感受到什麼情緒。選擇一項：

- 他接受了失去摯愛的事實，找回了平靜。
- 他決定去修復生命中一段破裂的關係。
- 他在接下來一個月內將變得相當開心快活。
- 他深深為這禮物著迷，無論如何都不願與它分開。
- 他把你視為親密的朋友來愛，並尋求更親密的關係。

導引者會決定這個禮物的形式，使其符合你為該生物所選擇的效果。
{{< /ability-block >}}

{{< ability-block title="侵入夢域" type="magician" >}}
{{< cost-tag number=5 >}}你觸碰一個睡著的 NPC（夢境主人）並進入他的夢境。你可以帶著任何跟你處於鄰近距離內的自願隊伍成員一起進入夢境。入侵夢境的成員將會進入一種出神狀態，無法感知到任何夢境之外的事物，即使受到傷害也一樣。

你決定你和你的同伴們會如何出現在夢境。你們可以在夢境世界中冒充其他人。一旦你們進入夢境，你們將會在夢境主人的腦海中創造出一個新的夢境。這個夢境有兩個場景。

**提議**. 在第一個場景，你們會遇到夢境主人並與他交談。由你決定這個場景的環境背景。<br/>
在這裡，你們可以嘗試說服夢境主人去做某件事。舉例來說，你們可以要求他放棄一項重大的追求、切斷與家人的聯繫、或是放棄他擁有的財富。

**試煉**. 在夢境接續的場景中，夢境主人最大的恐懼將會以怪物的形式出現，並試圖殺死他。導引者會根據你們提議的嚴重程度來選擇適合的怪物。如果你們嘗試戲劇性的改變夢境主人的生活，那個導引者可能會選擇讓擁有可怕能力的怪物登場。

這場戰鬥的規則與在夢境外的戰鬥相同，但你們不會死在這裡。所有進入夢境的玩家都會有全滿的生命值和１０點可用的 AP，並會在生命值歸零時立刻離開夢境。

如果你們擊敗怪物，那麼目標會在之後醒來時把這場夢當作是一個天啟。他將著手去執行你們建議的事。<br/>
如果怪物在夢境中殺死了夢境主人，則這個法術立即終止。所有人，包括夢境主人在內，都會醒過來。

你們在夢境中可以如同在現實世界一樣使用自己的能力，但所有這些效果都是虛假的，無法被帶離夢境世界之外。任何你們在夢境中花費的 AP 都會在你們離開時恢復。
